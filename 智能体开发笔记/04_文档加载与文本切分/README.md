# 04_文档加载与文本切分

本章节包含常见文档加载器和文本切分示例，适合准备 RAG、语义检索和文档问答数据。

文件说明：

- `静态网页加载器实战.py`：加载静态网页内容并进行切分。
- `PDF加载器实战.py`：演示 PDF 文档加载。
- `Word加载器实战.py`：演示 Word 文档加载。
- `多类型文档加载器实战.py`：支持多种文件类型混合加载。
- `文本切割实战.py`：基础文本切片与分段处理。
- `文本分割进阶实战.py`：更复杂的文本分割策略。
- `带优先级分隔符的文本分割器实战.py`：演示带优先级分隔符的切分规则。
- `提取PDF图片实战.py`：从 PDF 中提取图片及文本内容。
- `Langchian支持的文件加载器.py`：展示 LangChain 支持的常用加载器。

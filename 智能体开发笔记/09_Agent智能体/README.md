# 09_Agent智能体

- `REACT 智能体实战.py`：ReAct 风格 Agent 的实现。
- `Milvus向量库检索整合REACT智能体实战.py`：Agent 与 Milvus 检索结合。
- `create_agent新版Agent实战.py`：使用新版 create_agent 接口的示例。
- `个人AI助手智能体实战.py`：个人助手型 Agent 实战。
- `旧版initialize_agent迁移说明.md`：老版本 initialize_agent 迁移到新接口的注意事项。

# 03_输出解析与结构化输出

本章节关注大模型输出的稳定性、结构化解析与错误修复。

文件说明：

- `字符串解析器.py`：字符串格式输出解析示例。
- `JSON解析器.py`：JSON 输出解析示例。
- `PyDantic HelloWorld.py`：Pydantic 结构化输出基础示例。
- `PyDantic 进阶.py`：进一步介绍 Pydantic 输出解析技巧。
- `PyDantic再进阶.py`：更高级的 Pydantic 模型与错误处理示例。
- `将大模型结构化信息提取.py`：提取大模型结构化信息的实用方法。
- `调用大模型失败处理.py`：演示如何检测并重试解析失败的输出。
- `逗号分隔解析器并流式输出.py`：逗号分隔格式解析与流式输出结合。

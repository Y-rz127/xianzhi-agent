# 06_向量数据库_Milvus

本章节集中讲解 Milvus 向量数据库的创建、索引、DML、检索以及与 LangChain 集成。

文件说明：

- `Milvus向量数据库 创建Collection（表）实战.py`：创建集合和定义字段结构。
- `Milvus向量数据库索引实战（创建、查看、删除）.py`：索引管理与集合结构查看。
- `Milvus向量数据库DML实战（插入、删除、更新）.py`：向量数据的增删改操作。
- `Milvus向量数据库综合实战.py`：从创建到查询的整体流程演示。
- `LangChain  + Milvus 向量库实战（嵌入+增删数据）.py`：LangChain 与 Milvus 一体化示例。
- `LangChain + Milvus 向量检索进阶（MMR检索）.py`：向量检索与 MMR 检索优化。
- `Milvus创建带动态字段、分片表和数据库常用方法.py`：高级集合结构与分片场景。

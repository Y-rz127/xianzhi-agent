# 服务启动说明

本目录不强制提供 Docker Compose，因为不同机器上的 Milvus、Redis、Ollama 安装方式差异较大。运行相关脚本前按下面检查即可。

## Redis

用于 `10_记忆与多会话` 中的长期记忆示例。

```powershell
redis-cli ping
```

返回 `PONG` 表示可用。连接地址写入 `.env`：

```txt
REDIS_URL=redis://127.0.0.1:6379
```

## Milvus

用于 `06_向量数据库_Milvus`、`07_RAG检索与文档问答`、`09_Agent智能体`。

确认 `.env` 中有：

```txt
MILVUS_URI=http://127.0.0.1:19530
```

如果使用 Zilliz Cloud 或远程 Milvus，还需要设置 token。

## Ollama

用于本地嵌入模型示例。

```powershell
ollama list
```

`.env` 示例：

```txt
OLLAMA_BASE_URL=http://localhost:11434
OLLAMA_EMBEDDING_MODEL=nomic-embed-text
```

## MCP

先启动服务端，再运行客户端。示例路径：

```powershell
python ".\11_MCP协议\MCP 服务端实战.py"
```

# 01_基础调用与提示词

本章节示例聚焦于大模型调用、PromptTemplate、聊天提示和 LangSmith 入门。

文件说明：

- `最简单调用大模型.py`：最基础的大模型直接调用示例。
- `大模型的零样本（无示例）和少样本（带示例）提示工程.py`：演示零样本和少样本提示策略。
- `PromptTemplate HelloWorld.py`：介绍 LangChain PromptTemplate 的核心用法。
- `LLMChain HelloWorld.py`：演示 LLMChain 的简单链式调用。
- `流式输出 HelloWorld.py`：演示流式输出模型结果的方式。
- `多角色聊天提示词 模板.py`：构建多角色聊天提示模板。
- `多角色聊天提示词调用大模型.py`：实际调用大模型执行多角色聊天。
- `大模型接入LangSmith实战.py`：展示与 LangSmith 结合的实战流程。
- `LCEL表达式.py`：介绍 LCEL 表达式及其在提示工程中的应用。

# 运行环境说明

这些脚本主要用于学习 LangChain、Agent、RAG、Milvus、MCP 和记忆机制。它们依赖不同外部服务，不建议一次性全部运行。

## 推荐准备

1. 在 `PythonProject/AI` 下复制 `.env.example` 为 `.env`，填入自己的密钥和服务地址。
2. 运行脚本前先确认该脚本依赖的服务是否已启动。
3. 优先把脚本中的硬编码密钥和地址改成 `os.getenv(...)` 读取。
4. 每次只跑一个主题目录下的脚本，先理解输出，再继续下一个文件。

## 建议改造

- 新建统一配置模块，例如 `config.py`，集中读取 `.env`。
- 把演示数据、API Key、服务地址从脚本里抽离。
- 对 Tool 和 Agent 增加最小测试，避免工具 schema 改了以后 Agent 无法调用。
- 对 RAG 示例固定测试问题和期望证据，方便判断检索效果是否退化。

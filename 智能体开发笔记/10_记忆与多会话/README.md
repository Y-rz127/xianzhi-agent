# 10_记忆与多会话

本章节介绍短期记忆、长期记忆、多租户会话隔离和 Redis 持久化记忆示例。

文件说明：

- `短期记忆实战.py`：短期会话记忆的基础实现。
- `多会话隔离记忆.py`：演示会话隔离的记忆管理方式。
- `多租户 、多会话隔离记忆实战.py`：多租户与多会话隔离场景。
- `聊天模型嵌入历史记录实战.py`：使用嵌入向量构建对话历史检索。
- `长期记忆缓存到 Redis实战.py`：Redis 持久化长期记忆示例。
- `LangGraph store长期记忆实战.py`：使用 LangGraph store 存储长期记忆。
- `LangGraph checkpointer短期记忆实战.py`：使用 LangGraph checkpointer 管理短期记忆。
- `多轮对话内容的摘要生成与记忆管理实战.py`：结合摘要生成与记忆管理。
- `checkpointer与store区别.md`：解释 LangGraph checkpointer 与 store 的差异。

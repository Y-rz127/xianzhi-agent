# 02_LCEL链与Runnable编排

本章节演示 LangChain LCEL 语法和 Runnable 编排机制，用于构建更灵活的链路流程。

文件说明：

- `RunnablePassthrough的assign机制（动态添加字段）.py`：介绍如何使用 assign 动态扩展数据字段。
- `RunnableLambda高级实战.py`：展示 RunnableLambda 在复杂逻辑中的应用。
- `RunnableParallel并行链实战.py`：演示并行执行多个任务并合并结果。
- `RunnableBranch实现智能客服路由系统实战.py`：使用分支路由实现智能客服场景。

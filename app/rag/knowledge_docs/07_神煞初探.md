# 神煞初探

> 神煞为命理辅助星曜，主点睛、定性、应期。本篇收录全套高频实用神煞查法与意象。
> **核心原则：正五行、十神、格局、用神为核心；神煞仅作辅助参考。**

## 一、神煞使用边界

神煞起源于汉唐占星与择日，后融入命理。其作用：

- **辅助定性与应期**：神煞可点明人事性质（如桃花主情缘、驿马主奔波）与应期。
- **不可单凭断事**：神煞力量较弱，不可脱离正五行格局单独断吉凶。
- **主次分明**：神煞与正五行冲突时，以正五行为准。神煞只作补充印证。

## 二、吉神

### 1. 天乙贵人（最贵吉神）
主解危济困、逢凶化吉、贵人扶持。

**查法口诀**：
> 甲戊并牛羊，乙己鼠猴乡，丙丁猪鸡位，壬癸兔蛇藏，庚辛逢虎马，此是贵人方。

| 日干 | 天乙贵人 |
|------|----------|
| 甲、戊 | 丑、未 |
| 乙、己 | 子、申 |
| 丙、丁 | 亥、酉 |
| 壬、癸 | 卯、巳 |
| 庚、辛 | 寅、午 |

日干见贵人，主一生有贵人相助，逢凶化吉，危难有人救。

### 2. 禄神（食禄）
主衣食、自身有力。

**查法**：以日干查地支。
- 甲禄在寅，乙禄在卯，丙戊禄在巳，丁己禄在午，庚禄在申，辛禄在酉，壬禄在亥，癸禄在子。

日干见禄，主自身有力、衣食不缺、事业有根基。

### 3. 文昌
主学业、文采、考运。

**查法口诀**：甲乙巳午报君知，丙戊申宫丁己鸡，庚猪辛鼠壬逢虎，癸人见兔入云梯。

| 日干 | 文昌 |
|------|------|
| 甲 | 巳 |
| 乙 | 午 |
| 丙、戊 | 申 |
| 丁、己 | 酉 |
| 庚 | 亥 |
| 辛 | 子 |
| 壬 | 寅 |
| 癸 | 卯 |

### 4. 学堂、词馆
主学业、学问、文才。

**查法**：以日干（纳音长生为学堂，帝旺为词馆）。
- **学堂**（纳音长生位）：甲见亥、乙见午、丙戊见寅、丁己见酉、庚见巳、辛见子、壬见申、癸见卯。
- **词馆**（纳音帝旺位）：甲见卯、乙见寅、丙戊见午、丁己见巳、庚见酉、辛见申、壬见子、癸见亥。

日干见学堂词馆，主聪敏好学、文才出众、学业有成。

### 5. 红鸾、天喜（婚恋吉星）
主婚恋、喜事、姻缘。

**查法**：以年支查地支。
- **红鸾**：子年在卯，丑年在寅，寅年在丑，卯年在子，辰年在亥，巳年在戌，午年在酉，未年在申，申年在未，酉年在午，戌年在巳，亥年在辰。（口诀：从卯起子逆数）
- **天喜**：与红鸾对冲。即子年在酉，丑年在申，寅年在未，卯年在午，辰年在巳，巳年在辰，午年在卯，未年在寅，申年在丑，酉年在子，戌年在亥，亥年在戌。

年支或流年见红鸾天喜，主婚恋、添丁、喜事。

### 6. 天德贵人、月德贵人
主逢凶化吉、消灾解难。

**天德查法**（以月支查）：
- 正月（寅）见丁，二月（卯）见申，三月（辰）见壬，四月（巳）见辛，五月（午）见亥，六月（未）见甲，七月（申）见癸，八月（酉）见寅，九月（戌）见丙，十月（亥）见乙，十一月（子）见巳，十二月（丑）见庚。

**月德查法**（以月支查天干）：
- 寅午戌月见丙，申子辰月见壬，亥卯未月见甲，巳酉丑月见庚。

### 7. 魁罡
主刚强果断、掌权、武职。

**查法**：日柱为庚辰、庚戌、壬辰、戊戌四日之一者，为魁罡格。

## 三、凶煞

### 1. 羊刃（阳刃）
主刚烈、易折、好斗、伤灾。

**查法**：以日干查地支（帝旺位）。
- 甲刃在卯，丙戊刃在午，庚刃在酉，壬刃在子。（阴干无刃；或云乙刃在寅、丁己刃在巳、辛刃在申、癸刃在亥）

日干见羊刃，主刚强好斗、易伤灾、婚姻硬碰硬。身弱见刃反主身强有力，身旺见刃则过亢易折。

### 2. 驿马
主奔波、迁移、变动、出行。

**查法**：以年支或日支查地支。
- 申子辰马在寅，寅午戌马在申，巳酉丑马在亥，亥卯未马在巳。

见驿马主奔波、迁移、出行、变动。驿马逢冲主动荡远行。

### 3. 桃花（咸池）
主情缘、风流、人缘。

**查法**：以年支或日支查地支。
- 申子辰桃花在酉，寅午戌桃花在卯，巳酉丑桃花在午，亥卯未桃花在子。

见桃花主人缘好、异性缘、风流。墙内桃花（桃花在日支）主夫妻恩爱，墙外桃花（在年月时）主外遇风流。

### 4. 孤辰、寡宿
主孤独、婚迟、鳏寡。

**查法**：以年支查地支。
- 亥子丑年：孤辰在寅，寡宿在戌。
- 寅卯辰年：孤辰在巳，寡宿在丑。
- 巳午未年：孤辰在申，寡宿在辰。
- 申酉戌年：孤辰在亥，寡宿在未。

男命见孤辰主孤克，女命见寡宿主寡居，均主婚姻迟、夫妻缘薄。

### 5. 亡神、劫煞
主官非、口舌、损耗。

**查法**：以年支或日支查。
- 申子辰：亡神在亥，劫煞在巳。
- 寅午戌：亡神在巳，劫煞在亥。
- 巳酉丑：亡神在申，劫煞在寅。
- 亥卯未：亡神在寅，劫煞在申。

### 6. 空亡（旬空）
主虚耗、不实、缘薄、落空。

**查法**：以日柱（或年柱）所在旬，查旬空地支。

| 旬 | 十干十支 | 空亡 |
|------|----------|------|
| 甲子旬 | 甲子～癸酉 | 戌、亥 |
| 甲戌旬 | 甲戌～癸未 | 申、酉 |
| 甲申旬 | 甲申～癸巳 | 午、未 |
| 甲午旬 | 甲午～癸卯 | 辰、巳 |
| 甲辰旬 | 甲辰～癸丑 | 寅、卯 |
| 甲寅旬 | 甲寅～癸亥 | 子、丑 |

用神落空亡则用神虚浮无力，忌神落空亡则忌神减力。空亡逢冲合则解空，逢填实（岁运见之）则不空。

### 7. 天罗地网
主官非、病灾、困顿。

**查法**：
- 男命天罗在亥，地网在戌。
- 女命天罗在戌，地网在亥。
- 男忌戌亥，女忌亥戌。

命局或岁运见之，主官非、病灾、困顿、进退维谷。

### 8. 十恶大败
主破财、耗损、败业。

**查法**：日柱为以下十日之一者：
- 甲辰、乙巳、壬申、丙申、丁亥、庚辰、戊戌、癸亥、辛巳、己丑。

十恶大败日生，主财帛易散、事业易败，须节俭守成。

### 9. 华盖
主孤高、才艺、宗教、哲思。

**查法**：以年支或日支查地支。
- 申子辰华盖在辰，寅午戌华盖在戌，巳酉丑华盖在丑，亥卯未华盖在未。

见华盖主聪慧孤高、好文艺宗教、有玄学哲思之才，亦主孤独。

### 10. 三奇贵人
主奇才、异能、不凡。

**查法**：天干见以下三奇顺序（须按顺序，倒乱则不真）：
- **天上三奇**：甲、戊、庚
- **地下三奇**：乙、丙、丁
- **人中三奇**：壬、癸、辛

## 四、神煞综合应用要点

1. **主次分明**：正五行格局、十神、用神为论命核心，神煞仅辅助点睛。
2. **神煞为辅**：神煞可定性、应期，但不可单凭断吉凶。
3. **吉凶取用**：
   - 吉神（天乙、禄、文昌、红鸾天喜、天月德）扶用神则吉上加吉。
   - 凶煞（羊刃、亡神、劫煞、空亡、天罗地网）克用神则凶上加凶。
4. **神煞与五行主次**：神煞与正五行结论冲突时，以正五行为准。
5. **应期配合**：神煞多配合流年断应期。如红鸾天喜流年到主婚恋应期，驿马流年到主迁移应期。

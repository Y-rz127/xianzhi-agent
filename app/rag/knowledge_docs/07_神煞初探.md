# 神煞初探

神煞是八字中特殊的干支组合，用于辅助断命。

## 常见吉神
- 天乙贵人：逢凶化吉，贵人相助
- 文昌：学业聪明，文采出众
- 天德、月德：化解灾厄
- 驿马：迁动、出行、变化
- 桃花：人缘、感情（子午卯酉为桃花）

## 常见凶煞
- 羊刃：刚烈、易伤灾（刃为帝旺之地）
- 七杀：压力、险厄
- 亡神、劫煞：损耗、官非
- 孤辰寡宿：孤独

## 查法示例
- 天乙贵人：甲戊庚牛羊，乙己鼠猴乡，丙丁猪鸡位，壬癸蛇兔藏，六辛逢马虎
- 桃花：寅午戌见卯，巳酉丑见午，申子辰见酉，亥卯未见子

## 使用原则
神煞为辅，五行格局为主。神煞需结合命局五行旺衰判断，单独论神煞易误。
专业命理：以正五行、十神、用神为核心，神煞仅作辅助参考。

# 贫富层次判断

> 贫富层次为命理论命核心目标之一。本篇含富贵、中产、普通、劳碌、贫贱格局判定标准，财官印食伤组合分层次。
> 引用典籍：《渊海子平·论财官》《子平真诠·论格局高低》《滴天髓·论富贵》《神峰通考·论贫富》。
> **核心原则：格局高低定层次，用神有力定富贵；财官印食伤组合决定财富规模。**

## 一、富贵层次判定

### 1. 大富大贵
- **格局清透**：用神单一有力、无杂神混杂、无刑冲破害。
- **用神通根有力**：用神透干通根本气、紧贴日主。
- **财官印双全**：财生官、官生印、印生身，流通有情。
- **身旺任财官**：日主偏旺、能任财官。
- **岁运扶起用神**：大运行喜用运、扶起用神。
- **现实表现**：大富大贵、事业有成、地位显赫、财富巨万。

### 2. 中富中贵
- **格局尚清**：用神有力但稍有瑕疵。
- **用神透干但根稍弱**：或被轻冲轻克但有救应。
- **财官有力**：财官通根、但配合不完美。
- **身旺任财官但稍弱**：或身稍弱但有生扶。
- **岁运一般扶起**：大运平稳、有喜有忌。
- **现实表现**：中富中贵、事业稳定、地位中等、财富可观。

### 3. 小富小贵
- **格局有疵**：用神有力但有杂神混杂、或被刑冲破害。
- **用神透干虚浮**：透干但无根、或有制化但不完美。
- **财官有力但配合不佳**：或有比劫夺财、或官杀混杂。
- **身弱财多**：富屋贫人、为财奔波。
- **岁运有扶有破**：大运有喜有忌、起伏。
- **现实表现**：小富小贵、事业有起伏、地位一般、财富中等。

## 二、普通层次判定

### 1. 普通命
- **格局平庸**：用神无力、或被杂神混杂。
- **用神虚浮**：透干无根、或被合化变质。
- **财官无力**：财官虚浮、或被冲克。
- **身弱财多或身旺财弱**：难以任财。
- **岁运平平**：大运无大扶无大破。
- **现实表现**：普通命、事业平平、地位普通、财富勉强温饱。

### 2. 劳碌命
- **格局破败**：用神被克伤无救应、或杂神混杂无制。
- **用神受伤**：被冲克合化、或入墓逢空。
- **财官被克**：比劫夺财、伤官见官。
- **身弱无依**：日主偏弱、无印比生扶。
- **岁运多破**：大运多忌神、少喜用。
- **现实表现**：劳碌命、事业难成、奔波劳碌、财富微薄。

### 3. 贫贱命
- **格局完全破败**：用神被克尽无救应。
- **用神废**：用神被合化变质、或被冲克入墓。
- **财官被克尽**：比劫夺财无制、伤官见官无救。
- **身弱无依或身旺无依**：偏枯之命。
- **岁运全破**：大运全忌神、无喜用。
- **现实表现**：贫贱命、事业无成、贫困潦倒、灾厄多。

## 三、财富规模判定

### 1. 巨富
- **偏财为用通根透干**：经商有道、横财可得。
- **食伤生财为用**：以技艺、经营致富。
- **财星逢生合岁运**：进财应期。
- **身旺任财**：能承大财。
- **现实**：财富巨万、亿万富翁。

### 2. 富
- **正财为用通根透干**：稳定薪资、积蓄丰足。
- **食伤生财为用**：经营有道、财源广进。
- **身旺任财**：能承财。
- **现实**：财富百万千万、富裕。

### 3. 中产
- **财星为用但力稍弱**：或有比劫夺财但有制。
- **食伤生财但生扶不力**：财源有但规模有限。
- **身旺任财但财不大**：能承财但财不大。
- **现实**：财富几十万至百万、中产。

### 4. 温饱
- **财星中和**：财星不旺不弱、勉强温饱。
- **身弱财多**：富屋贫人、为财奔波。
- **比劫夺财**：财来财去、难以积蓄。
- **现实**：财富勉强温饱、收支平衡。

### 5. 贫困
- **财星被克尽**：比劫夺财无制、或财星入墓逢空。
- **身弱无依**：难以任财。
- **岁运多破财**：大运多比劫夺财、忌神运。
- **现实**：贫困、财富微薄、负债。

## 四、地位规模判定

### 1. 大贵
- **官星为用清透通根**：仕途顺遂、地位显赫。
- **官印相生**：有学历有地位、仕途双美。
- **财官双美**：有财力有地位。
- **身旺任官**：能承官威。
- **现实**：高官显贵、地位显赫。

### 2. 贵
- **官星为用通根**：仕途有成、地位中等。
- **官印相生**：有学历有地位。
- **身旺任官**：能承官。
- **现实**：中高官、地位较高。

### 3. 中产地位
- **官星为用但力稍弱**：仕途有波折、地位中等。
- **官星中和**：地位普通。
- **身旺任官但官不大**：能承官但官不大。
- **现实**：中层干部、地位中等。

### 4. 普通地位
- **官星虚浮或为忌**：仕途难成、地位普通。
- **官杀混杂**：仕途反复、是非多。
- **伤官见官**：仕途受阻、与领导冲突。
- **现实**：普通职员、地位普通。

### 5. 低地位
- **官星被克尽**：仕途无成、地位低。
- **七杀无制**：灾厄、压力。
- **身弱官杀重**：受压制、地位低。
- **现实**：底层、地位低、受压制。

## 五、贫富层次与格局配合

| 格局 | 财富层次 | 地位层次 |
|------|----------|----------|
| 正官格（清透有印护） | 中富以上 | 大贵 |
| 七杀格（有食制有印化） | 中富以上 | 大贵 |
| 正财格（通根有护） | 富 | 中贵 |
| 偏财格（通根有食生） | 巨富 | 中贵 |
| 正印格（有官生） | 中富 | 贵 |
| 偏印格（有制化） | 中产 | 中贵 |
| 食神格（有财泄） | 富 | 中产 |
| 伤官格（有财泄或有印制） | 富 | 中产 |
| 从财格 | 巨富 | 中贵 |
| 从杀格 | 中富 | 大贵 |
| 专旺格 | 中富以上 | 大贵 |
| 化气格（成） | 大富大贵 | 大贵 |

## 六、贫富层次与岁运

### 1. 扶起用神岁运
- 大运行喜用运：格局扶起、发福。
- 流年行喜用运：当年顺遂、进财、升迁。
- 岁运扶起用神：财富地位提升。

### 2. 破格岁运
- 大运行忌神运：格局破、败。
- 流年行忌神运：当年逆阻、破财、降职。
- 岁运破格：财富地位下降。

### 3. 救应岁运
- 大运行救应运：格局救应、转机。
- 流年行救应运：当年逢凶化吉。
- 岁运救应：逢凶化吉、先难后易。

## 七、贫富层次与流年应期

### 1. 进财应期
- 财星逢生合岁运：进财。
- 食伤生财岁运：经营进财。
- 财库逢冲开库岁运：大进财。
- 身弱者逢印比生扶岁运：能任财则进财。

### 2. 破财应期
- 比劫夺财岁运：破财。
- 财星被冲克岁运：破财。
- 财星入墓岁运：财被压。
- 财星逢空亡岁运：虚财、回款落空。

### 3. 升迁应期
- 官星逢生合岁运：升迁。
- 官印相生岁运：仕途顺。
- 用神逢生扶岁运：整体提升。

### 4. 降职应期
- 官星被冲克岁运：降职。
- 伤官见官岁运：仕途受阻。
- 官杀混杂岁运：是非多、仕途反复。

## 八、回答口径

- 先说贫富层次：大富大贵？中富中贵？普通？劳碌？贫贱？
- 再说财富规模：巨富？富？中产？温饱？贫困？
- 最后说地位规模：大贵？贵？中产？普通？低？
- 引经据典佐证，结论先给白话，再引古籍原文。
- 不说"必富""必贵"，要给条件："若……则……"。

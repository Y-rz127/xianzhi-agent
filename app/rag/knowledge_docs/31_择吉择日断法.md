# 择吉择日断法

> 八字命理择日择吉、选黄道吉日、开业嫁娶搬迁动土择日的专项断法规则。供择吉择日 Worker 检索引用。

## 一、总则

1. **以用事人八字喜用神为根基**：所选日课干支五行宜助旺用神、避开忌神。
2. **事项定用神侧重**：开业重财官、嫁娶重合婚、搬迁重印比安稳、动土重印星护身。
3. **避凶煞**：避开与用事人年命刑冲、三煞、月破、四离四绝等凶日。
4. **选吉神当值**：天德、月德、天赦、三合、六合等吉神值日优先。

## 二、用神择日

### 1. 按用神定日课五行

- **用神为木**：宜甲乙寅卯日、亥子日（水生木）。
- **用神为火**：宜丙丁巳午日、寅卯日（木生火）。
- **用神为土**：宜戊己辰戌丑未日、巳午日（火生土）。
- **用神为金**：宜庚辛申酉日、辰戌丑未日（土生金）。
- **用神为水**：宜壬癸亥子日、申酉日（金生水）。

### 2. 忌神日避

- 忌神当值之日宜避：如忌火则避丙丁巳午日。
- 忌神被制化时，可酌情使用。

## 三、事项择日

### 1. 开业
- 重财官：宜财星、官星当值，忌比劫夺财。
- 选天赦、天愿、益后日。
- 避月破、大耗、劫煞日。

### 2. 嫁娶
- 重合婚：宜新郎新娘八字用神互补之日。
- 选天德、月德、三合、六合、天喜日。
- 避月破、破碎、孤辰寡宿日。
- 忌与双方年命刑冲。

### 3. 搬迁
- 重印比安稳：宜印星、比劫当值，忌伤官见官。
- 选天赦、天恩、母仓日。
- 避月破、月厌、四废日。

### 4. 动土
- 重印星护身：宜印星当值，忌七杀攻身。
- 选天德、月德、天赦日。
- 避土府、土符、地囊日。

### 5. 出行
- 重驿马与贵人：宜驿马、天乙贵人日。
- 避往亡、归忌日。

## 四、吉神

### 1. 天德贵人
- 主万事吉利、逢凶化吉。

### 2. 月德贵人
- 主慈祥善良、事宜顺遂。

### 3. 天赦
- 主百无禁忌、万事皆宜，尤宜祈福、动土。

### 4. 三合 / 六合
- 三合（申子辰、寅午戌、巳酉丑、亥卯未）：主和合成事。
- 六合（子丑、寅亥、卯戌、辰酉、巳申、午未）：主和谐顺遂。

### 5. 天喜 / 红鸾
- 主婚恋喜庆，嫁娶择日优先。

## 五、凶煞

### 1. 月破
- 月支冲日，主破败、不宜举事。

### 2. 三煞
- 申子辰月煞巳午未、寅午戌月煞亥子丑、巳酉丑月煞寅卯辰、亥卯未月煞申酉戌。
- 三煞方不宜动土、搬迁。

### 3. 四离四绝
- 四离：春分、夏至、秋分、冬至前一日。
- 四绝：立春、立夏、立秋、立冬前一日。
- 主气机交接、不宜大事。

### 4. 往亡 / 归忌
- 主出行不利、忌远行。

### 5. 大耗 / 劫煞
- 主破财、忌开业开仓。

## 六、与年命关系

1. **日课不冲年命**：所选日支不冲用事人年支。
2. **日课不刑年命**：所选日支不刑用事人年支。
3. **日课合年命**：日支合年支为大吉，主事成人和。

## 七、择日流程参考

1. 排定用事人八字，确认用神喜忌。
2. 确定事项类型（开业/嫁娶/搬迁/动土/出行）。
3. 在事项对应的吉日内筛选助旺用神之日。
4. 排查凶煞：避开月破、三煞、四离四绝等。
5. 复核与年命关系：不冲不刑、宜合。
6. 综合权衡，给出 2-3 个候选吉日。

## 八、合规口径

1. 择日为传统民俗参考，重大事宜（医疗、法律、安全）务必以专业意见为准。
2. 不绝对断"某日必大凶""某日必大吉"，应说明依据与变数。
3. 不宣扬择日改运、择日保平安等绝对化表述。
4. 涉及医疗手术、法律诉讼等，明确提示以医院、律师专业意见为优先。

# 择吉择日断法

> 八字命理择日择吉、选黄道吉日、开业嫁娶搬迁动土择日的专项断法规则。供择吉择日 Worker 检索引用。

## 一、总则

1. **以用事人八字喜用神为根基**：所选日课干支五行宜助旺用神、避开忌神。
2. **事项定用神侧重**：开业重财官、嫁娶重合婚、搬迁重印比安稳、动土重印星护身。
3. **避凶煞**：避开与用事人年命刑冲、三煞、月破、四离四绝等凶日。
4. **选吉神当值**：天德、月德、天赦、三合、六合等吉神值日优先。

## 二、用神择日

### 1. 按用神定日课五行

- **用神为木**：宜甲乙寅卯日、亥子日（水生木）。
- **用神为火**：宜丙丁巳午日、寅卯日（木生火）。
- **用神为土**：宜戊己辰戌丑未日、巳午日（火生土）。
- **用神为金**：宜庚辛申酉日、辰戌丑未日（土生金）。
- **用神为水**：宜壬癸亥子日、申酉日（金生水）。

### 2. 忌神日避

- 忌神当值之日宜避：如忌火则避丙丁巳午日。
- 忌神被制化时，可酌情使用。

## 三、十二建星

十二建星为传统择日基础，每日轮值一颗，吉凶不同：

| 建星 | 吉凶 | 宜 | 忌 |
|------|------|----|----|
| 建 | 平 | 出行、上任 | 开仓、动土 |
| 除 | 吉 | 除旧布新、扫舍、治病 | 嫁娶、远行 |
| 满 | 吉 | 祭祀、祈福、结亲 | 栽种 |
| 平 | 平 | 修造、涂泥 | 嫁娶、开渠 |
| 定 | 吉 | 嫁娶、开市、冠笄 | 诉讼、出行 |
| 执 | 平 | 修造、祭祀 | 搬迁、出行 |
| 破 | 凶 | 破屋、坏垣 | 诸事皆忌 |
| 危 | 凶 | 祭祀 | 嫁娶、出行、登高 |
| 成 | 吉 | 嫁娶、开市、入学、纳采 | 诉讼 |
| 收 | 吉 | 纳财、收获、入宅 | 开市、出行 |
| 开 | 吉 | 开市、出行、嫁娶、上任 | 埋葬 |
| 闭 | 凶 | 埋葬 | 开市、出行 |

### 建星用法

- 吉日（满、定、成、收、开）宜举大事。
- 凶日（破、危、闭）忌举大事。
- 平日（建、除、平、执）可做小事、不宜大事。
- 建星与用神五行结合：吉日+用神当值=上吉。

## 四、事项择日

### 1. 开业
- 重财官：宜财星、官星当值，忌比劫夺财。
- 选天赦、天愿、益后日。
- 避月破、大耗、劫煞日。
- 建星宜：成、开、满日。
- 具体建议：选财星当值+建星吉日+无冲年命之日。

### 2. 嫁娶
- 重合婚：宜新郎新娘八字用神互补之日。
- 选天德、月德、三合、六合、天喜日。
- 避月破、破碎、孤辰寡宿日。
- 忌与双方年命刑冲。
- 建星宜：成、定、满日。
- 具体建议：选天喜/红鸾日+双方用神当值+无冲年命之日。

### 3. 搬迁
- 重印比安稳：宜印星、比劫当值，忌伤官见官。
- 选天赦、天恩、母仓日。
- 避月破、月厌、四废日。
- 建星宜：成、开、收日。
- 具体建议：选印星当值+建星吉日+不冲日支之日。

### 4. 动土
- 重印星护身：宜印星当值，忌七杀攻身。
- 选天德、月德、天赦日。
- 避土府、土符、地囊日。
- 建星宜：成、开日。
- 具体建议：选印星当值+无三煞方+建星吉日。

### 5. 出行
- 重驿马与贵人：宜驿马、天乙贵人日。
- 避往亡、归忌日。
- 建星宜：开、成日。
- 具体建议：选驿马当值+天乙贵人日+建星吉日。

### 6. 签约
- 重官印：宜官星、印星当值，忌伤官见官。
- 选天德、月德、三合日。
- 避月破、劫煞日。
- 建星宜：定、成日。

### 7. 考试
- 重印星：宜印星当值，忌食伤泄气。
- 选文昌日、天乙贵人日。
- 建星宜：成、开日。

### 8. 入宅
- 重印比安稳：宜印星、比劫当值。
- 选天恩、母仓日。
- 避月破、四废日。
- 建星宜：收、成日。

## 五、吉神

### 1. 天德贵人
- 主万事吉利、逢凶化吉。
- 查法：正丁二坤宫、三壬四辛同、五亥六甲上、七癸八艮逢、九丙十居乙、子巳丑庚中。

### 2. 月德贵人
- 主慈祥善良、事宜顺遂。
- 查法：寅午戌月丙、申子辰月壬、亥卯未月甲、巳酉丑月庚。

### 3. 天赦
- 主百无禁忌、万事皆宜，尤宜祈福、动土。
- 查法：春戊寅、夏甲午、秋戊申、冬甲子。

### 4. 三合 / 六合
- 三合（申子辰、寅午戌、巳酉丑、亥卯未）：主和合成事。
- 六合（子丑、寅亥、卯戌、辰酉、巳申、午未）：主和谐顺遂。

### 5. 天喜 / 红鸾
- 主婚恋喜庆，嫁娶择日优先。
- 天喜查法：年支逆推（子年酉、丑年申、寅年未……）。
- 红鸾查法：年支顺推（子年卯、丑年辰、寅年巳……）。

### 6. 文昌
- 主聪明好学，考试择日优先。
- 查法：甲乙巳午报、丙戊申宫找、丁己鸡同窝、庚辛猪鼠位、壬癸寅卯期。

### 7. 驿马
- 主出行顺利，出行择日优先。
- 查法见方位迁移断法。

## 六、凶煞

### 1. 月破
- 月支冲日，主破败、不宜举事。

### 2. 三煞
- 申子辰月煞巳午未、寅午戌月煞亥子丑、巳酉丑月煞寅卯辰、亥卯未月煞申酉戌。
- 三煞方不宜动土、搬迁。

### 3. 四离四绝
- 四离：春分、夏至、秋分、冬至前一日。
- 四绝：立春、立夏、立秋、立冬前一日。
- 主气机交接、不宜大事。

### 4. 往亡 / 归忌
- 主出行不利、忌远行。

### 5. 大耗 / 劫煞
- 主破财、忌开业开仓。

### 6. 月厌
- 主不宁、忌嫁娶搬迁。

### 7. 土府 / 土符 / 地囊
- 主土煞、忌动土修造。

### 8. 孤辰寡宿
- 主孤独、忌嫁娶。

## 七、与年命关系

1. **日课不冲年命**：所选日支不冲用事人年支。
2. **日课不刑年命**：所选日支不刑用事人年支。
3. **日课合年命**：日支合年支为大吉，主事成人和。
4. **日课生年命**：日干生年干为吉，主顺遂。
5. **日课克年命**：日干克年干为凶，主不顺。

## 八、择日流程参考

1. 排定用事人八字，确认用神喜忌。
2. 确定事项类型（开业/嫁娶/搬迁/动土/出行/签约/考试/入宅）。
3. 在事项对应的吉日内筛选助旺用神之日。
4. 查建星：选吉日（满、定、成、收、开）。
5. 排查凶煞：避开月破、三煞、四离四绝等。
6. 复核与年命关系：不冲不刑、宜合。
7. 综合权衡，给出 2-3 个候选吉日。

## 九、常见事项择日案例

### 案例1：开业择日
- 用事人：甲子年生、用神为火
- 步骤：选丙丁巳午日（火日助用神）→ 查建星选成/开日 → 避月破劫煞 → 避子午冲年命 → 综合选丙午成日。

### 案例2：嫁娶择日
- 用事人：男辛未年、女癸酉年
- 步骤：选天喜/红鸾日 → 选双方用神当值日 → 避冲双方年支（未丑冲、酉卯冲）→ 选三合六合日 → 综合选吉日。

### 案例3：搬迁择日
- 用事人：壬申年生、用神为木
- 步骤：选甲乙寅卯日（木日助用神）→ 查建星选收/成日 → 避寅申冲年命 → 选天恩日 → 综合选吉日。

## 十、合规口径

1. 择日为传统民俗参考，重大事宜（医疗、法律、安全）务必以专业意见为准。
2. 不绝对断"某日必大凶""某日必大吉"，应说明依据与变数。
3. 不宣扬择日改运、择日保平安等绝对化表述。
4. 涉及医疗手术、法律诉讼等，明确提示以医院、律师专业意见为优先。
5. 择日结果仅供参考，最终以用户自主判断为准。

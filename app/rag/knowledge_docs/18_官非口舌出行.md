# 官非口舌出行

> 官非口舌、牢狱、驿马出行为人生常见问事。本篇含官非判断、牢狱风险、驿马出行、异地发展、搬迁应期。
> 引用典籍：《渊海子平·论官非》《三命通会·论官符》《滴天髓·论刑冲》。
> **核心原则：官杀混杂、伤官见官、三刑齐现为官非主因；驿马逢冲为出行搬迁主因。**

## 一、官非口舌断法

### 1. 官非主因
- **官杀混杂**：正官七杀同现且无制，主事多反复、是非不断、官非。
- **伤官见官**：伤官克正官，主口舌官非、与领导冲突、仕途受阻。
- **三刑齐现**：寅巳申三刑、丑未戌三刑，主官非、刑伤、牢狱。
- **七杀无制**：七杀重身轻无制化，主官非、灾厄。
- **羊刃冲克**：羊刃逢冲，主血光、官非、刑伤。
- **天罗地网**：男忌戌亥，女忌亥戌，主官非牵绊、行运难展。

### 2. 官非轻重
- **官杀混杂+三刑**：大官非、牢狱。
- **伤官见官+羊刃**：口舌官非、刑伤。
- **七杀无制+财党杀**：官非破财、重灾。
- **有印化杀或有食制杀**：官非减轻、逢凶化吉。

### 3. 官非应期
- **官杀混杂逢引动岁运**：当年有官非。
- **伤官见官岁运**：当年口舌官非。
- **三刑逢引动岁运**：当年刑伤官非。
- **七杀逢生扶岁运**：当年官非重。
- **羊刃逢冲岁运**：当年血光官非。
- **天罗地网岁运**：当年官非牵绊。

### 4. 官非化解
- **有印化杀**：贵人相助、逢凶化吉。
- **有食制杀**：以智化解、转危为安。
- **有财生官**：以财通关、化解官非。
- **有比劫抗杀**：朋友相助、共渡难关。
- **岁运过去**：官非运过则灾减。

## 二、牢狱断法

### 1. 牢狱主因
- **官杀混杂无制+三刑齐现**：牢狱之灾。
- **七杀重身轻+羊刃冲克**：刑伤牢狱。
- **伤官见官+官星被克尽**：仕途官非、牢狱。
- **财星被劫尽+官杀克身**：因财生灾、牢狱。
- **三刑齐现+羊刃**：刑伤牢狱。

### 2. 牢狱应期
- **官杀混杂+三刑逢引动岁运**：当年牢狱。
- **七杀逢生扶+羊刃逢冲岁运**：当年刑伤牢狱。
- **伤官见官+官星被克尽岁运**：当年仕途官非、牢狱。
- **财星被劫尽+官杀克身岁运**：当年因财生灾、牢狱。

### 3. 牢狱轻重
- **三刑齐现+官杀混杂+羊刃**：大牢狱、重刑。
- **有印化杀或有食制杀**：牢狱减轻、逢凶化吉。
- **岁运过去**：牢狱运过则出狱。

### 4. 牢狱化解
- **有印化杀**：贵人相助、减轻刑罚。
- **有食制杀**：以智化解、转危为安。
- **有财生官**：以财通关、减轻刑罚。
- **岁运过去**：牢狱运过则出狱。

## 三、小人是非断法

### 1. 小人主因
- **官杀混杂**：小人多、是非不断。
- **伤官见官**：与领导冲突、口舌是非。
- **比劫旺为忌**：同事朋友背后中伤、争财。
- **七杀逢生扶**：小人攻身、压力。
- **刑冲破害多**：人际冲突、是非多。

### 2. 小人应期
- **官杀混杂逢引动岁运**：当年小人多。
- **比劫夺财岁运**：当年同事朋友争财、背后中伤。
- **伤官见官岁运**：当年与领导冲突、口舌。
- **七杀逢生扶岁运**：当年小人攻身。

### 3. 小人化解
- **有印化杀**：贵人相助、化解小人。
- **有食制杀**：以智化解、远离小人。
- **有比劫抗杀**：朋友相助、共渡难关。

## 四、驿马出行断法

### 1. 驿马查法
- **驿马查法**：以年支或日支查地支。
  - 申子辰：驿马在寅。
  - 亥卯未：驿马在巳。
  - 寅午戌：驿马在申。
  - 巳酉丑：驿马在亥。

### 2. 驿马主事
- **驿马逢冲**：出行、迁移、奔波、变动。
- **驿马逢合**：出行受阻、暂缓。
- **驿马逢生扶**：出行顺利、得利。
- **驿马逢流年引动**：当年出行、迁移。
- **驿马逢大运引动**：十年出行、迁移多。

### 3. 驿马出行类型
- **驿马+财星**：出行求财、经商、贸易。
- **驿马+官杀**：出行任官、升迁、调动。
- **驿马+印星**：出行求学、考学、升迁。
- **驿马+食伤**：出行旅游、社交、创作。
- **驿马+比劫**：出行合伙、朋友同行。

### 4. 驿马出行应期
- **驿马逢冲岁运**：当年出行、迁移。
- **驿马逢合岁运**：当年出行受阻、暂缓。
- **驿马逢生扶岁运**：当年出行顺利。

## 五、异地发展断法

### 1. 异地发展适合
- **驿马逢冲**：适合异地发展、奔波求财。
- **年柱逢冲**：远方发展、离祖成家。
- **日支逢冲**：异地婚姻、分居。
- **财星在年柱或时柱**：远方求财、远方之妻。
- **官星在年柱或时柱**：远方任职、远方之夫。

### 2. 异地发展不顺
- **驿马逢合**：出行受阻、暂缓。
- **驿马逢冲克**：出行多变、奔波劳碌。
- **日支逢冲**：异地婚姻不稳、分居。

### 3. 异地发展应期
- **驿马逢冲岁运**：当年异地发展。
- **年柱逢冲岁运**：当年远方发展。
- **流年合日支**：当年回乡、团聚。

## 六、搬迁断法

### 1. 搬迁主因
- **驿马逢冲**：搬迁、迁移。
- **辰戌冲、丑未冲**：房产动、土地动、搬迁。
- **日支逢冲**：搬家、居住变动。
- **印星逢冲**：房产变动、文书变动。

### 2. 搬迁应期
- **驿马逢冲岁运**：当年搬迁。
- **辰戌冲、丑未冲岁运**：当年房产动、搬迁。
- **日支逢冲岁运**：当年搬家、居住变动。
- **印星逢冲岁运**：当年房产变动。

### 3. 搬迁吉凶
- **搬迁为喜用**：搬迁顺利、得利。
- **搬迁为忌神**：搬迁不顺、破财、纠纷。

## 七、官非口舌与格局配合

| 格局 | 官非特征 | 化解 |
|------|----------|------|
| 正官格 | 官星清透则少官非；伤官见官则口舌官非 | 印制伤护官 |
| 七杀格 | 杀重无制则官非重；杀有制则官非轻 | 食制杀、印化杀 |
| 伤官格 | 伤官见官则口舌官非；伤官有制则无碍 | 印制伤、财化伤 |
| 食神格 | 食神制杀则官非轻；枭神夺食则官非重 | 财制枭护食 |
| 财格 | 财星被劫则破财官非；财星有护则无碍 | 官制比劫护财 |

## 八、回答口径

- 先说官非风险：有无官非？轻重？
- 再说应期：何时有官非？何时化解？
- 最后说建议：如何化解？如何规避？
- 引经据典佐证，结论先给白话，再引古籍原文。
- 不说"必坐牢""必无官非"，要给条件："若……则……"。

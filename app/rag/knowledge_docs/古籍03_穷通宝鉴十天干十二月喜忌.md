# 古籍03：《穷通宝鉴》十天干十二月喜忌

> 《穷通宝鉴》（又名《拦江网》）为调候用神实战手册，专论十天干在十二个月的寒暖燥湿喜忌。本篇录其核心调候法则，为取用神之第一标准。
> 适用场景：调候用神、寒暖燥湿、分月喜忌。
> 分层调用优先级：用神调候 → 《穷通宝鉴》。
> **核心原则：调候为先，寒命暖之，燥命润之，同一天干不同月份喜忌迥异。**

## 一、调候总论

### 1. 调候核心原则
- **寒命宜暖**：冬月生人（亥子丑月），命局寒湿，宜丙火暖之。
- **燥命宜润**：夏月生人（巳午未月），命局燥热，宜癸水润之。
- **春木旺宜泄**：春月生人（寅卯辰月），木旺当令，宜丙火泄秀。
- **秋金旺宜炼**：秋月生人（申酉戌月），金旺当令，宜丁火炼金。

### 2. 调候优先原则
- 当格局用神与调候用神冲突时，**调候优先**。
- 如冬月水旺，虽格局取官杀为用，但无丙火暖身则格局难显，须先调候。
- 调候用神不损害格局用神时，两者并用。

### 1. 三春甲木（寅卯辰月）
- **总论**：三春甲木，花繁叶茂，先用丙火泄秀，次用癸水润根。

- **三月甲木（辰月）**：木气将竭，先取庚金，次取壬水。庚金克木成材，壬水泄庚润木。

- **五月甲木（午月）**：木性枯焦，专取癸水为要。无癸则木枯，丁火助之。
- **六月甲木（未月）**：木气已衰，癸水润之，丁火助之。

### 3. 三秋甲木（申酉戌月）
- **总论**：三秋甲木，木气凋零，宜水养不宜火焚。

- **八月甲木（酉月）**：金旺克木，宜丁火制金，癸水润木。
- **九月甲木（戌月）**：土旺木弱，宜甲木助之，癸水润之。

### 4. 三冬甲木（亥子丑月）

# 五行生克关系

> 本篇阐明五行生克正反规律、通关调和、五行万象及真假旺衰，是判断命局平衡失衡的核心。
> 引用典籍：《滴天髓·理气》《渊海子平·论五行生克》。

## 一、五行相生

木生火，火生土，土生金，金生水，水生木。循环不息。

相生含义：
- 生我者为印（正印、偏印），主庇护、学业、母亲。
- 我生者为食伤（食神、伤官），主才华、子女、泄秀。

**生之真伪**：相生须有源、有力。母弱子强则母被泄过度（如水弱木旺，水被木吸干）；母强子弱则子被淹没（如水多木漂）。

## 二、五行相克

木克土，土克水，水克火，火克金，金克木。

相克含义：
- 克我者为官杀（正官、七杀），主约束、事业、丈夫（女命）。
- 我克者为财（正财、偏财），主钱财、妻父（男命）。
- 同我者为比劫（比肩、劫财），主竞争、兄弟、分财。

## 三、反生反克（五行亢害）——核心难点

五行过亢则反常，"物极必反"。此为《滴天髓》理气之要。

### 1. 反生为克（生多反克）
- **木能生火，火多木焚**：木生火，火过旺反烧木（身弱火多之命，印被泄尽）。
- **火能生土，土多火晦**：火生土，土过厚反埋火（火被土晦，主人晦暗无光）。
- **土能生金，金多土变**：土生金，金过旺反耗土变质（土被金泄，主虚耗）。
- **金能生水，水多金沉**：金生水，水过旺反沉金（金被水溺，主沉沦不显）。
- **水能生木，木多水缩**：水生木，木过旺反吸干水（水被木夺，主源枯）。

### 2. 反克（克者反被克）
- **金能克木，木坚金缺**：金克木，木过坚反缺金（身弱木旺，官杀反损）。
- **木能克土，土重木折**：木克土，土过重反折木（身弱土旺，比劫被埋）。
- **土能克水，水多土荡**：土克水，水过旺反冲垮土（水旺土崩，主堤防失守）。
- **水能克火，火多水竭**：水克火，火过旺反蒸干水（火旺水干，主水枯）。
- **火能克金，金多火熄**：火克金，金过多反压灭火（金多火少，火被压灭）。

### 3. 反生（克极反生）
亢极则反。如：金旺至极则生水，水极旺反润金（金水相生）；火极则生土，土极反护火。此乃"亢则害，承乃制"之理。

### 4. 通关调和
两行交战，需第三行调和通关，方化克为生。

**通关用法**：
- **金木交战**（金克木）：取水通关，金生水、水生木，化克为生。
- **木土交战**（木克土）：取火通关，木生火、火生土。
- **土水交战**（土克水）：取金通关，土生金、金生水。
- **水火交战**（水克火）：取木通关，水生木、木生火。
- **火金交战**（火克金）：取土通关，火生土、土生金。

**通关成立条件**：
1. 通关之神须有根有力（透干通根），否则无力调和。
2. 通关之神不被冲克合化。
3. 通关之神位置居中（紧贴两方最佳）。
4. 两方力量不可过分悬殊（一方极弱则无通关之必要）。

**两组五行交战通关**：如金木交战又水火交战，需看何者为主、何者为辅，取主要矛盾通关。多重交战者，多主一生波折。

## 五、五行真假旺衰区分

判断旺衰不能只看数量，须辨真假虚实。此为《滴天髓》"真假旺衰"之精微。

### 1. 真旺
- 当令得时（月令为同类五行）
- 党众有力（地支三合三会、多干透出）
- 有生有扶（印比助身）
- 无克无泄或克泄有制
真旺者身强，可任财官，喜克泄耗。

### 2. 假旺（似旺实弱）
- 虽多党众，但皆虚浮无根（天干比劫多，地支无根）
- 或党众被冲克合化（如木多而申酉金局克之）
- 或生于不当令之月，徒有其表
假旺者表面旺实则虚，不可任财官，反喜生扶。误判为旺则取用必错。

### 3. 真弱
- 失令无根（月令克泄，地支无同类根气）
- 克泄交加（官杀食伤多而旺）
- 无生无扶（无比劫印星，或印被克）
真弱者身弱，喜印比生扶，忌克泄耗。

### 4. 假弱（似弱实旺）
- 虽少而精，通根本气有力（如甲木生于申月，但地支寅卯辰会木局）
- 或得月令暗助（藏干透出）
- 或克泄之神反被克（官杀被食伤制、食伤被印制）
假弱者表面弱实则强，反喜克泄耗。误判为弱则取用必错。

### 5. 虚旺
- 旺而无制（一行独旺，无克无泄，反成偏枯）
- 旺极无依（旺而无财官可任，反主孤独劳碌）
虚旺者旺而不实，多主表面风光、内里空虚。

### 6. 实弱
- 弱而有根（虽弱而通根，根气未拔）
- 弱而有救（弱而有印生扶，或克泄之神受制）
实弱者虽弱而有救，逢生扶之运则发。

**判断要点**：旺衰须看月令、根气、党众、生扶克泄综合判定，不可单凭数量。古云："得令者旺，失令者衰；有根者实，无根者虚。"

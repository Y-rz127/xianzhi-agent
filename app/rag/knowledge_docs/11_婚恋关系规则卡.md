# 婚恋关系规则卡

> 婚恋为用户高频问事之一。本篇含男女命财官混杂分断、桃花正缘暧昧区分、复合分手异地晚婚二婚完整判断、结婚应期三重条件。
> 引用典籍：《渊海子平·论六亲》《三命通会·论婚姻》《神峰通考·论夫妻》《滴天髓·论夫妻》。

## 一、感情判断核心

感情问题不能只看桃花，要同时看日支、配偶星、合冲刑害、大运流年。

- **男命**：以财星为伴侣缘分的重要观察点。正财为正妻，偏财为偏房情人或再婚之妻。
- **女命**：以官杀为伴侣缘分的重要观察点。正官为正夫，七杀为偏房情人或再婚之夫。
- **日支为配偶宫**：日支坐喜用神、与日主相生相合，主夫妻和睦；日支坐忌神、逢冲逢刑，主婚姻不稳。
- **配偶星透藏与有情**：配偶星透出且与日主相生相合（如日主甲木见壬水生扶，财星己土合甲），同时大运承接，关系更容易落地。
- **桃花引动**：桃花被流年引动，常见于社交变多、暧昧、被关注，但未必等于稳定关系。
- 不可脱离全局强弱单独断婚：身弱财多之男、身弱官杀混杂之女，皆主为情所累、难以掌控婚姻。

## 二、男命财星混杂分断

男命财星即配偶星，财星组合直接关系婚姻质量。

### 1. 财星混杂定义
- 正财与偏财同现于命局（不论天干透出或地支藏干），即为财星混杂。
- 财星多而旺（三个及以上）：情感经历丰富，易有多段感情、多婚之象。

### 2. 财星混杂吉凶
- **财星为喜用而混杂**：异性缘佳、桃花旺、左右逢源，但易因情生事。
- **财星为忌而混杂**：为情所累、破财因色、婚姻反复、多婚之命。
- **正偏财并透且无制**：明暗双妻、外遇明显、婚姻难稳。
- **比劫旺而夺混杂财**：配偶易被夺、离婚再娶、合伙分妻。

### 3. 财星位置与婚姻应期
- 财星在年柱：早婚、远方之妻、长辈介绍。
- 财星在月柱：适婚年龄得配、同事同学、父母之命。
- 财星在日支：妻星得位、夫妻亲近、婚姻稳定（喜用则吉）。
- 财星在时柱：晚婚、子息缘、后半生之伴。

### 4. 财星入墓、逢空、被克
- 财星入墓（辰戌丑未）：妻缘薄、妻病、妻离散。
- 财星逢空亡：缘来缘去、虚缘假象、迟婚。
- 财星被比劫克夺：离异、争妻、配偶外遇。
- 财星被冲：夫妻分居、矛盾激化、关系变化。

## 三、女命官杀混杂分断

女命官杀即配偶星，官杀组合直接关系婚姻质量。

### 1. 官杀混杂定义
- 正官与七杀同现于命局（不论天干透出或地支藏干），即为官杀混杂。
- 官杀多而旺（三个及以上）：异性缘旺但难专一，易有多段感情。

### 2. 官杀混杂吉凶
- **官杀为喜用而混杂**：有魅力、多追求者、事业感情双丰收，但易陷入选择困难。
- **官杀为忌而混杂**：为情所困、婚姻多变、感情波折、易遇渣男。
- **正偏官并透且无制**：明暗双夫、外遇明显、婚姻反复。
- **七杀无制化而旺**：配偶强势、易受压制、感情伤害重、遇人不淑。

### 3. 官杀位置与婚姻应期
- 官星在年柱：早婚、远方之夫、长辈介绍。
- 官星在月柱：适婚年龄得配、工作环境相识。
- 官星在日支（官星得位）：夫缘近、婚姻稳（喜用则吉）。
- 官星在时柱：晚婚、老夫少妻、后半生伴侣。

### 4. 官杀入墓、逢空、被克
- 官星入墓：夫缘薄、夫病、夫离散、丧夫之兆。
- 官星逢空亡：姻缘虚、迟婚、缘来缘去。
- 官杀被伤官克破：伤官见官、口舌争吵、离婚之兆。
- 官杀被冲：夫妻矛盾、分居、婚姻变化。

## 四、日支逢冲逢刑婚姻矛盾细化

日支为配偶宫，日支受伤直接关系婚姻稳定。

### 1. 日支逢冲
- **流年冲日支**：当年婚姻动、有变化（结婚、分手、分居、怀孕、搬家）。
- **大运冲日支**：十年婚姻不稳、反复变化。
- **原局冲日支**：先天婚姻不稳、迟婚或多次婚姻之象。
- **近冲（月支或时支冲日支）**：力量大，明显变化。
- **远冲（年支冲日支）**：力量小，暗动、隐性问题。

### 2. 日支逢刑
- **日支参与三刑**：夫妻矛盾激烈、官非连累、家暴之兆。
- **日支自刑（辰午酉亥）**：内耗、冷战、自我折磨、自残倾向。
- **日支逢刑+七杀/羊刃**：家暴、刑伤、配偶灾。

### 3. 日支逢害
- 隐性隔阂、沟通不畅、貌合神离。
- 比直接冲刑温和，但长期损耗。

### 4. 日支逢合
- **合为喜用**：夫妻和睦、有贵人撮合、婚姻稳定。
- **合为忌**：配偶被合走、外遇、分居。
- **日支被流年合**：当年有婚姻动象（结婚或外遇）。

## 五、单身遇桃花、暧昧、正缘区分

桃花有真假、虚实、正偏之分，须细辨。

### 1. 桃花分类
- **墙内桃花（年支见桃花）**：夫妻恩爱、感情美满、内桃花主吉。
- **墙外桃花（日时见桃花）**：外遇、风流、多情、易惹桃花是非。
- **真桃花（申子辰见酉、亥卯未见子等正桃花）**：异性缘真实、感情落实。
- **滚浪桃花（日时柱同见且财官并旺）**：感情泛滥、难以专一。

### 2. 暧昧与正缘区分
- **暧昧（偏财/七杀为用但虚浮无根）**：关系不明确、有缘无分、拖泥带水。
- **正缘（正财/正官为用且通根透干）**：关系明确、稳定、可落地、走向婚姻。
- **流年桃花（桃花逢流年引动）**：当年异性缘旺，但是否成正缘看配偶星是否同动。
- **大运桃花（桃花逢大运引动）**：十年异性缘旺，感情周期长。

### 3. 桃花吉凶
- **桃花为喜用**：人缘佳、得异性助、感情顺利。
- **桃花为忌**：因色生灾、名誉受损、感情纠纷。
- **桃花逢冲**：感情变化、分手、桃花破。
- **桃花逢合**：感情落实、结婚、桃花结。
- **桃花逢空亡**：虚缘、单恋、有花无果。

## 六、复合、分手、异地婚姻、晚婚、二婚完整判断

### 1. 复合判断
问复合时，优先看当前流年是否合回配偶宫、是否解冲，以及双方现实阻力是否降低。

- **可复合条件**：
  - 流年合日支、合解原冲（如原局日支逢冲，流年合来解冲）。
  - 配偶星逢生扶岁运、配偶星回位。
  - 比劫或官杀压力减轻、现实阻力降低。
  - 红鸾天喜被引动且配偶星同动。

- **难复合条件**：
  - 流年冲日支、刑日支、克配偶星。
  - 比劫夺财岁运（男）、伤官见官岁运（女）。
  - 配偶星入墓逢空、配偶与他人合化。
  - 现实阻力大（异地、家庭反对、经济问题未解）。

- **建议口径**：复合看双方意愿+现实条件+命局岁运信号，三者皆吉方可成。

### 2. 分手判断
- **流年冲动配偶宫**：分手、关系变化。
- **比劫夺财岁运（男）**：第三者介入、配偶被夺。
- **伤官见官岁运（女）**：口舌争吵、关系破裂。
- **配偶星入墓逢空**：感情无果、缘尽。
- **三刑岁运**：刑伤分手、矛盾激化。

### 3. 异地婚姻
- **驿马星动（驿马逢冲）**：异地、远嫁、奔波。
- **配偶星在年柱或时柱**：远方之配、远嫁或晚婚。
- **日支逢冲**：分居、异地工作。
- **桃花逢驿马**：异地恋、远距离感情。
- **建议**：异地婚姻须看大运是否最终合聚，若大运合日支、解原冲，终能团聚。

### 4. 晚婚
- **官杀/财星迟现（时柱）**：晚婚之命。
- **配偶星逢空亡**：迟婚、缘来晚。
- **日支逢冲原局**：婚姻不稳、迟婚可解。
- **食伤过旺（女）/比劫过旺（男）**：克配偶星、迟婚为宜。
- **建议**：晚婚命局宜先立业后成家，早婚易反复。

### 5. 二婚判断
- **原局正偏财混杂（男）/正偏官混杂（女）且无制**：明暗双妻、二婚之命。
- **日支逢冲且配偶星被克**：离异再娶/再嫁。
- **比劫旺夺财（男）/伤官旺克官（女）**：婚姻反复、二婚。
- **配偶星入墓逢冲开库**：丧偶再婚。
- **三刑在日支**：刑伤配偶、再婚。
- **建议**：二婚命局宜晚婚、宜找同样有婚史者、宜异地婚姻化解。

## 七、结婚应期判定标准

结婚应期须满足以下三重条件之一或组合：

### 1. 流年合配偶宫
- 流年地支合日支（如日支酉，流年辰合酉）。
- 流年地支解原冲（如原局日支卯逢酉冲，流年戌合卯解冲）。
- 流年地支合化配偶星（如男命日主甲木，流年己土合甲，己为正财）。

### 2. 引动配偶星
- **男命**：流年引动财星（财星逢生、逢合、透出、通根）。
- **女命**：流年引动官星（官星逢生、逢合、透出、通根）。
- **配偶星在大运流年同现**：婚期信号强烈。
- **配偶星逢生合岁运**：当年有结婚机会。
- **配偶星从墓库中透出**：婚期，库中财官出。

### 3. 红鸾天喜触发
- **红鸾星动**：流年地支为红鸾星（如年支申子辰人，流年见卯为红鸾）。
- **天喜星动**：流年地支为天喜星（如年支申子辰人，流年见酉为天喜，红鸾天喜互为对宫）。
- 红鸾天喜主婚恋、添丁、喜庆，引动配偶宫与配偶星则婚期至。
- 神煞仅作辅助，须与配偶宫、配偶星同动方可断婚。

### 5. 结婚应期综合判断
- **大运提供婚姻土壤**（大运喜用、配偶星逢生扶）。
- **流年引动配偶宫与配偶星**（合日支、引动配偶星）。
- **红鸾天喜或桃花同动**（神煞辅助）。
- **三重条件同时满足**：婚期最明确。
- **两重条件满足**：有机会，须看现实配合。
- **一重条件满足**：仅桃花动、或仅合日支，未必结婚，可能只是恋爱或暧昧。

## 八、六亲关系与婚姻连带

婚姻不只看配偶星，还要看六亲关系对婚姻的影响。

- **父母宫（月柱）冲日支**：父母干涉婚姻、婆媳矛盾、岳婿关系紧张。
- **子女宫（时柱）冲日支**：因子女问题影响夫妻关系。
- **比劫旺（男）**：兄弟朋友夺财、配偶易被横刀夺爱。
- **食伤旺（女）**：子女克夫、伤官见官、夫受压制。
- **印星过旺**：长辈过度保护、配偶受压制、婆媳矛盾。
- **官杀混杂带桃花（女）**：异性缘旺易招惹是非、婚姻反复。

## 九、婚姻质量层次

| 婚姻类型 | 命局特征 | 现实表现 |
|----------|----------|----------|
| 上等婚姻 | 日支坐喜用、配偶星为用且通根、无冲刑 | 夫妻和睦、相互扶持、白头偕老 |
| 中等婚姻 | 日支不坐喜用但无冲刑、配偶星中和 | 平淡稳定、小矛盾可化解 |
| 下等婚姻 | 日支逢冲刑、配偶星为忌或被克夺 | 反复争吵、分合、离婚、丧偶 |
| 多婚之命 | 正偏财混杂无制（男）/官杀混杂无制（女）、日支逢冲刑 | 二婚、三婚、感情泛滥 |
| 克配偶 | 比劫夺财（男）/伤官见官（女）、羊刃冲日支 | 配偶灾病、丧偶、刑伤 |

## 十、回答口径

- 用户问感情时，先回答"有没有机会、稳定性如何、该主动还是观望"。
- 不要说"必婚""必离""一定复合"，要表达为"机会更明显""稳定性不足""需要现实条件配合"。
- 必要时追问：是否已有对象、关系阶段、是否异地、是否有现实阻力。
- 依据控制在 2-4 条：日支状态、配偶星强弱、大运流年触发、神煞辅助。
- 引经据典佐证，结论先给白话，再引古籍原文。

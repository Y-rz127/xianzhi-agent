# 起名改名规则

> 八字命理起名改名的核心原则、五行补益方法、常用汉字五行属性、字形字义与音律取舍，以及命名合规口径。供起名改名 Worker 检索引用。

## 一、起名总则

1. **以八字用神喜忌为根本**：名字的五行属性应补益命主用神、喜神，避开忌神。起名不是套用固定吉字，而是因人而异、因局而异。
2. **五行补益优先于字形字义**：先确保五行方向正确，再在符合五行的字中挑选寓意好、音律和的字。
3. **起名仅为文化民俗参考**：名字对人生起调候补益之辅助作用，不保证改运、不替代个人努力与现实决策。

## 二、五行补益原则

### 1. 按日主强弱定补法

- **身弱**：补生扶日主之五行——印（生我）、比（同我）。如日主甲木弱，喜水（印）木（比），名字宜含水木字。
- **身强**：宜泄耗——食伤（我生）、财（我克）、官杀（克我）。如日主甲木强，喜火（食伤）土（财）金（官），名字宜含火土金字。
- **中和偏旺/偏弱**：以用神为补，兼顾调候。

### 2. 按用神喜忌定补法

- **用神为火**：名字宜含火属性字（炎、煜、灿、辉、明、旭）。
- **用神为水**：宜含水属性字（涵、泽、浩、淼、润、清）。
- **用神为木**：宜含木属性字（林、森、栋、柏、荣、茂）。
- **用神为金**：宜含金属性字（鑫、铭、锋、锦、钰、铃）。
- **用神为土**：宜含土属性字（坤、坦、城、培、磊、垣）。
- **忌神之五行尽量回避**：如忌火则避免炎、煜等火字。

### 3. 调候优先

- **寒命（冬生水旺）**：喜火调候，名字宜含火字暖局。
- **燥命（夏生火旺）**：喜水润局，名字宜含水字降燥。
- 调候与用神冲突时，以调候为优先参考。

## 四、字形字义取舍

1. **寓意积极**：选字义吉祥、向上的字，避开贬义、凶煞、消极字（如"凶、厄、衰、亡、孤"等）。
2. **字形稳重**：字形结构匀称，避免过于生僻难写难认的字。
3. **不与长辈重字**：传统避讳，避免与直系长辈同名同字。
4. **避开谐音不雅**：注意整名连读的谐音，避免产生负面联想（如"杜子腾"谐音"肚子疼"）。
5. **男女有别**：男名宜刚毅宏大（刚、伟、强、杰、豪），女名宜柔美温婉（婉、婷、雅、静、丽）；现代可灵活，但避免性别混淆过大。

## 五、音律要求

1. **声调搭配**：名字各字声调（平仄）宜错落，避免三字同声调导致平淡拗口。
   - 平声：一声、二声；仄声：三声、四声。
   - 推荐组合：平仄平、仄平仄、平平仄、仄仄平等错落型。
2. **声母韵母避免重复**：相邻字声母或韵母相同易拗口（如"李丽立"声母全 l）。
3. **响亮度**：开口呼字（a、o、e 起头）较响亮，适合放在末字提升气势。
4. **整体韵律**：连读顺口，无突兀停顿或拗口音。

## 六、命名禁忌

1. **忌用凶煞字**：凶、厄、煞、杀、死、亡、病、灾等。
2. **忌用贬义字**：劣、愚、弱、败、贫、贱等。
3. **忌用过于生僻字**：影响登记、社交、识别。
4. **忌与先人重名**：传统孝道避讳。
5. **忌谐音不雅**：见上文。

## 七、命名合规口径

1. **起名改名仅为文化民俗参考**，不保证改运、不替代医疗/法律/教育等专业建议。
2. **不宣传改运逆天改命**：名字起调候补益之辅助作用，不可夸大为改变命运的工具。
3. **最终以家长意愿与户籍规定为准**：命理建议仅供参考，不凌驾于家庭决定与法律法规。
4. **涉及未成年**：起名建议应客观中立，不制造焦虑。
5. **表述规范**：需表达调整作用时，使用"五行补益/平衡/调候"等表述，避免出现"改运/改命/逆天改命"等字眼。

## 八、起名流程参考

1. 排定八字，确认日主强弱与用神喜忌。
2. 确定需补五行方向（用神/喜神/调候）。
3. 在对应五行常用字中筛选寓意好、字形稳、音律和的字。
4. 组合两字或三字名，检查声调、谐音、整体韵律。
5. 与家长沟通确认，兼顾家庭意愿与户籍规定。

## 九、示例：用神补益取名

- **日主壬水身弱，用神金水**：宜补金水。可选"铭涵"（铭金+涵水）、"钰泽"（钰金+泽水）、"鑫浩"（鑫金+浩水）。
- **日主丙火身强，用神土金**：宜泄耗补土金。可选"坦锋"（坦土+锋金）、"磊铭"（磊土+铭金）、"城钰"（城土+钰金）。
- **冬生水旺寒命，调候喜火**：宜火暖局。可选"煜明"、"灿辉"、"炜旭"。

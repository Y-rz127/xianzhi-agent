# 起名改名规则

> 八字命理起名改名的核心原则、五行补益方法、常用汉字五行属性、字形字义与音律取舍、三才五格简述、改名注意事项，以及命名合规口径。供起名改名 Worker 检索引用。

## 一、起名总则

1. **以八字用神喜忌为根本**：名字的五行属性应补益命主用神、喜神，避开忌神。起名不是套用固定吉字，而是因人而异、因局而异。
2. **五行补益优先于字形字义**：先确保五行方向正确，再在符合五行的字中挑选寓意好、音律和的字。
3. **起名仅为文化民俗参考**：名字对人生起调候补益之辅助作用，不保证改运、不替代个人努力与现实决策。

## 二、五行补益原则

### 1. 按日主强弱定补法

- **身弱**：补生扶日主之五行——印（生我）、比（同我）。如日主甲木弱，喜水（印）木（比），名字宜含水木字。
- **身强**：宜泄耗——食伤（我生）、财（我克）、官杀（克我）。如日主甲木强，喜火（食伤）土（财）金（官），名字宜含火土金字。
- **中和偏旺/偏弱**：以用神为补，兼顾调候。

### 2. 按用神喜忌定补法

- **用神为火**：名字宜含火属性字（炎、煜、灿、辉、明、旭）。
- **用神为水**：宜含水属性字（涵、泽、浩、淼、润、清）。
- **用神为木**：宜含木属性字（林、森、栋、柏、荣、茂）。
- **用神为金**：宜含金属性字（鑫、铭、锋、锦、钰、铃）。
- **用神为土**：宜含土属性字（坤、坦、城、培、磊、垣）。
- **忌神之五行尽量回避**：如忌火则避免炎、煜等火字。

### 3. 调候优先

- **寒命（冬生水旺）**：喜火调候，名字宜含火字暖局。
- **燥命（夏生火旺）**：喜水润局，名字宜含水字降燥。
- 调候与用神冲突时，以调候为优先参考。

### 4. 五行补益层次

1. **第一优先**：补用神五行（最关键）。
2. **第二优先**：补喜神五行（辅助用神）。
3. **第三优先**：调候补益（寒暖燥湿）。
4. **避免**：忌神五行（尽量不用）。

## 四、三才五格简述

### 1. 三才

三才指天格、人格、地格的五行属性组合：

- **天格**：姓氏笔画+1（单姓）或姓氏总笔画（复姓），代表祖上运。
- **人格**：姓氏末字+名字首字笔画，代表主运（最重要）。
- **地格**：名字总笔画+1（单名）或名字总笔画（双名），代表前运。

### 2. 五格

- **天格**：祖上运，不可改。
- **人格**：主运，最关键，宜生不宜克。
- **地格**：前运（36岁前），宜生不宜克。
- **外格**：副运，看社交人际。
- **总格**：后运（36岁后），宜吉不宜凶。

### 3. 三才吉凶

- **大吉**：天人生、人地生（如木火土）。
- **吉**：天人比和、人地比和（如木木火）。
- **凶**：天人克、人地克（如木金木）。
- **大凶**：天人克+人地克（如金木金）。

### 4. 取名时三才五格的使用

- 三才五格仅作辅助参考，不作为起名的唯一依据。
- 五行补益优先于三才五格。
- 若五行补益与三才五格冲突，以五行补益为准。
- 三才五格的计算以康熙字典笔画为准。

## 五、字形字义与音律

### 1. 字形取舍

- 宜选结构匀称、笔画适中的字（8-16画为佳）。
- 避免生僻字、异体字、繁难字。
- 避免与长辈同字、同音。
- 避免字形不雅（如"死""病"等偏旁）。

### 2. 字义取舍

- 宜选寓意积极、吉祥、美好的字。
- 避免贬义、消极、不雅的字。
- 避免过于张扬、浮夸的字（如"霸""帝"）。
- 避免过于柔弱、消极的字（如"弱""悲"）。

### 3. 音律取舍

- 声调搭配：平仄相间，避免全平或全仄。
- 韵母搭配：避免同韵母连用（如"明明"）。
- 声母搭配：避免同声母连用（如"波波"）。
- 谐音避讳：避免不雅谐音（如"杜子腾"→肚子疼）。
- 整体读感：朗朗上口、抑扬顿挫。

### 4. 性别区分

- 男名宜阳刚：刚、强、伟、杰、豪、磊、峰、鹏、飞、翔。
- 女名宜柔美：芳、兰、婷、雅、慧、颖、琳、瑶、萱、薇。
- 中性名可通用：晨、睿、涵、泽、瑞、宁、安、然、清、逸。

## 六、改名注意事项

### 1. 改名时机

- 大运交接之年：改名效果最佳，新旧运交替时改名可助新运。
- 流年走用神运：改名助旺用神，效果明显。
- 人生重大转折：升学、就业、结婚等节点改名。
- 不宜在忌神运改名：改名方向可能被忌神影响。

### 2. 改名后的过渡期

- 改名后1-3年为过渡期，运势逐渐转变。
- 新名使用频率越高，效果越明显。
- 日常称呼、签名、社交名尽量使用新名。
- 户籍改名需走法律程序，非必须但效果最强。

### 3. 改名不改运的误区

- 改名仅为辅助调候补益，不保证改运。
- 改名不能替代个人努力和现实行动。
- 改名效果因人而异，与八字格局、大运流年配合有关。
- 改名后仍需积极行动，不能坐等好运。

### 4. 改名与原名的关系

- 新名五行应补原名之不足。
- 新名不宜与原名五行相冲。
- 新名可保留原名中五行有利的字。
- 改名不必全改，可改一字。

## 七、起名流程参考

1. 排定命主八字，确认用神喜忌。
2. 确定五行补益方向（补用神、喜神、调候）。
3. 在对应五行字库中筛选寓意好、音律和的字。
4. 组合搭配，检查三才五格（辅助参考）。
5. 检查谐音、字形、字义是否合适。
6. 给出 2-3 个候选名字，说明五行补益依据。

## 八、合规口径

1. 起名改名仅为文化民俗参考，不保证改运、不替代个人努力。
2. 不宣扬"改名必改运""好名必好运"等绝对化表述。
3. 最终以家长意愿和户籍规定为准，命理建议仅供参考。
4. 不涉及封建迷信、符咒法事等非理性内容。
5. 涉及户籍改名，提示以当地公安机关规定为准。

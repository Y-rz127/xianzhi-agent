# 事业财运规则卡

> 事业财运为用户最高频问事。本篇含八格事业适配、跳槽创业副业细分断法、财运分层、求财应期与回款风险。
> 引用典籍：《渊海子平·论财官》《子平真诠·论格局》《神峰通考·论财》。

## 一、事业判断核心

事业工作优先看官杀、印星、食伤与日主强弱的配合。

- **官杀**：规则、职位、压力、上级、平台约束。官杀清透且有印化，主稳定平台、职级晋升、专业资质路线。
- **食伤**：表达、技术输出、产品能力、自由度。食伤生财时，适合用技能、内容、产品、销售转化收入。
- **印星**：学习、证书、贵人、制度资源。日主偏弱时，印星能承压，适合先借平台和资质发展。
- **比劫**：同辈、竞争、合伙、团队。比劫旺而财弱时，合伙与分钱问题要谨慎。
- **财星**：收入、资源、目标。财为养命之源，官为荣身之本。

## 二、八格事业适配

不同格局，事业方向适配不同：

| 格局 | 事业适配 | 不宜 |
|------|----------|------|
| 正官格 | 公职、行政、管理、体制内、大型机构稳定岗 | 自由散漫、抗上越权 |
| 七杀格 | 武职、军警、执法、开拓型岗位、危机管理 | 文牍清闲、被动执行 |
| 正财格 | 稳定薪资、财务、银行、实业经营、守成 | 投机冒险、横财 |
| 偏财格 | 经商、贸易、投资、销售、外交、流通业 | 死工资、坐班固定 |
| 正印格 | 文职、教育、学术、文化、宗教、研究、文秘 | 商业争利、投机 |
| 偏印格 | 偏门、玄学、技术专才、设计、五术、科研 | 大众管理、销售 |
| 食神格 | 文艺、餐饮、教育、服务、医护、福利、子女相关 | 严肃执法、争权 |
| 伤官格 | 技艺、演艺、销售、写作、发明、自由职业、营销 | 体制内守规矩、服从 |

## 三、跳槽、裸辞、合伙创业、投资、开店、副业细分断法

### 1. 跳槽
- 问跳槽不只看流年，还要看当前大运是否支持职业变化。
- 流年冲动月柱、官杀、事业相关十神时，易出现岗位变化、领导调整、组织重组。
- 食伤旺而财星被引动，适合主动争取机会、谈资源、做项目转化。
- 官杀压力大而印星不足时，不建议裸辞，宜先确定岗位、直属领导、合同和现金流。
- 大运行喜用神运：跳槽得利、升迁。
- 大运行忌神运：跳槽多败，宜稳守。

### 2. 裸辞
- 财星有根、食伤生财、印比护身：可裸辞过渡，再寻良机。
- 财星虚浮、官杀克身、比劫夺财：不宜裸辞，断流则困。
- 七杀旺无制、羊刃冲克：裸辞多灾，宜骑驴找马。

### 3. 合伙创业
- 看财星、食伤、比劫和大运承载力。
- 财星被冲或比劫夺财明显时，投资和合伙要保守。
- 比劫旺为用：合伙得力，团队互助。
- 比劫旺为忌：合伙争财、内讧分赃，不宜合伙。
- 食伤生财为用：创业有成，以技艺致富。
- 官杀混为忌：创业多官非，宜守不宜创。

### 4. 投资
- 偏财为用且通根：投资有道，横财可得。
- 偏财为忌或虚浮：投资多败，宜守不宜攻。
- 比劫夺财岁运：忌大额投资，防破财。
- 财星逢生合岁运：投资应期，可适度布局。

### 5. 开店
- 食伤生财、财星通根、印星护身：开店有成。
- 财星入墓、官杀克身：开店多耗、易倒闭。
- 驿马逢冲：开店宜流动、连锁、异地。

### 6. 副业
- 食伤旺为用：副业有成，多源收入。
- 印旺克食伤：副业转化不足，宜先提升技能。
- 比劫旺：副业宜合伙分摊。

## 四、财运分层

财星不是越旺越好，必须看日主能否承财。

### 1. 正财稳定收入
- 正财通根、日主能任：稳定薪资、积蓄丰足。
- 正财为用且无伤：财源稳、收入可观。
- 正财为忌或身弱财多：富屋贫人，钱来钱去、为财奔波。

### 2. 偏财投资横财
- 偏财为用通根：经商有道、横财可得、意外之财。
- 偏财为忌虚浮：投机多败、财来财去。
- 偏财逢冲合：财源起伏大。

### 3. 比劫夺财破财
- 比劫旺为忌、财星被克：破财、夺妻、争产、合伙内讧。
- 比劫夺财岁运：大破财、投资失败、被坑骗。

### 4. 食伤生财变现
- 食伤生财为用：以技艺、内容、产品、销售致富，经营有道。
- 食伤生财逢生扶：财源广进。
- 食伤被枭夺：断财源、变现受阻。

### 5. 印旺无财收入微薄
- 印旺克食伤、财星无根：想法多而转化少、收入微薄、依赖他人。
- 印多为忌：懒惰无成、守成依赖。

## 五、求财应期与回款风险

### 1. 求财应期
- 财星逢生合岁运：进财应期。
- 食伤生财岁运：经营进财。
- 财库逢冲开库岁运：库中财出，大进财。
- 日主逢印比生扶岁运（身弱者）：能任财则进财。

### 2. 回款风险
- 财星被冲：回款难、财来财去。
- 财星入墓：财被压、回款拖延。
- 财星被劫：回款被截、被坑骗。
- 财星逢空亡：虚财、空头支票、回款落空。

### 3. 负债判断
- 比劫夺财岁运：易负债、借贷难还。
- 财多身弱岁运：因财生灾、负债。
- 官杀克身岁运：因官非破财、负债。

### 4. 官非破财
- 官杀混杂岁运：官非破财。
- 伤官见官岁运：口舌官非破财。
- 三刑岁运：刑伤破财。
- 羊刃冲克岁运：血光破财。

## 六、回答口径

- 用户问事业财运时，先回答"能不能动、适不适合、风险在哪里"。
- 依据控制在 2-4 条：当前大运、目标流年、日主强弱、财官食印关系。
- 不要说"必定发财""一定升职"，要给条件：有 offer、现金流、合伙协议、岗位稳定性。
- 结合格局适配给方向建议，引经据典佐证。

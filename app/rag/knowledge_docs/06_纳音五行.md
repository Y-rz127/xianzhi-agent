# 纳音五行

> 纳音五行是六十甲子配三十种五行，每两干支配一纳音。本篇专注纳音意象、合婚、格局及与正五行的主次边界。
> 引用典籍：《三命通会·纳音》《李虚中命书》。
> 完整六十甲子纳音表见 [01_天干地支基础.md](01_天干地支基础.md)。

## 一、纳音概述

纳音五行是六十甲子配三十种五行，每两干支配一纳音。纳音源于汉代律吕、五行数理，由鬼谷子创立、李虚中用以论命。

完整六十甲子纳音表见 01_天干地支基础.md。三十组纳音分金木水火土五类，每类六组：

### 纳音分类
- **金六**：海中金、剑锋金、白蜡金、沙中金、金箔金、钗钏金
- **木六**：大林木、杨柳木、松柏木、平地木、桑柘木、石榴木
- **水六**：涧下水、泉中水、长流水、天河水、大溪水、大海水
- **火六**：炉中火、山头火、霹雳火、山下火、覆灯火、天上火
- **土六**：路旁土、城头土、屋上土、壁上土、大驿土、沙中土

## 二、纳音意象详解

### 1. 金六
- **海中金**（甲子乙丑）：金匿海中，宝气内藏。主才深不露、待时而发，性情沉静，早年难显，中年发。
- **剑锋金**（壬申癸酉）：金已成器，利刃出匣。主刚锐果断、有胆识，宜武职军警，性急易折。
- **白蜡金**（庚辰辛巳）：金初成器，光华含蓄。主温润华贵、外柔内刚，性贵气而好美。
- **沙中金**（甲午乙未）：金散沙中，需淘洗方显。主散财聚财、起伏大，须磨炼方成。
- **金箔金**（壬寅癸卯）：金薄贴物，华而不实。主外表光鲜、内里虚浮，依附贵人而显。
- **钗钏金**（庚戌辛亥）：金为首饰，柔美装饰。主阴柔华贵、好美，女命尤贵。

### 2. 木六
- **大林木**（戊辰己巳）：林木茂盛，栋梁之材。主茂盛、成片、根基深，性厚道稳重。
- **杨柳木**（壬午癸未）：柳柔随风，婀娜多姿。主柔顺善变、依人，性灵活而外向。
- **松柏木**（庚寅辛卯）：松柏长青，岁寒不凋。主坚贞长寿、高洁，性刚直有节。
- **平地木**（戊戌己亥）：林木平地而生，朴实无华。主踏实平常、福寿安康，性平和。
- **桑柘木**（壬子癸丑）：桑柘养蚕，有用之材。主有实用、勤劳，性朴实有德。
- **石榴木**（庚申辛酉）：石榴多子，花木有果。主多子多福、晚成，性贞洁。

### 3. 水六
- **涧下水**（丙子丁丑）：山涧清泉，清澈低浅。主清纯内秀、低位而洁，性安静。
- **泉中水**（甲申乙酉）：泉涌不绝，源源而来。主源远流长、有德，性温和勤勉。
- **长流水**（壬辰癸巳）：江河长流，源远流长。主流通不息、奔波，性活络远行。
- **天河水**（丙午丁未）：天上甘露，尊贵清润。主高贵纯洁、福泽深厚，性尊贵。
- **大溪水**（甲寅乙卯）：溪水奔流，喧腾不息。主流荡不羁、好动，性外向多变。
- **大海水**（壬戌癸亥）：汪洋浩瀚，包容万象。主胸怀宽广、大度，性雄浑能容。

### 4. 火六
- **炉中火**（丙寅丁卯）：炉火炽盛，炼物成器。主性急燥烈、有炼物之能，性刚而烈。
- **山头火**（甲戌乙亥）：山火燎原，势不可挡。主声势浩大、易兴易灭，性外显易怒。
- **霹雳火**（戊子己丑）：雷霆之火，迅猛暴烈。主突发变动、迅猛，性急而奇。
- **山下火**（丙申丁酉）：山下萤火，光华含蓄。主内秀含光、晚发，性温而隐。
- **覆灯火**（甲辰乙巳）：灯火覆照，照暗为明。主指引启迪、文秀，性温和有智。
- **天上火**（戊午己未）：日光普照，光明正大。主光明磊落、尊贵显赫，性高洁外露。

### 5. 土六
- **路旁土**（庚午辛未）：路旁载物，卑微柔顺。主谦和载物、随和，性卑顺有德。
- **城头土**（戊寅己卯）：城墙高耸，护城守土。主厚重守卫、信实，性稳重可靠。
- **屋上土**（丙戌丁亥）：屋瓦覆盖，遮蔽风雨。主庇护安居、家业，性顾家厚道。
- **壁上土**（庚子辛丑）：墙壁立立，隔内御外。主隔绝防护、内向，性谨慎有守。
- **大驿土**（戊申己酉）：驿站通衢，往来辐辏。主流通通达、奔走，性好客广交。
- **沙中土**（丙辰丁巳）：沙土松散，无定无依。主漂泊不定、易散，性流动多变。

## 三、纳音合婚细则

纳音合婚为古法年命合婚，以年柱纳音（命主）论婚配吉凶。

### 1. 纳音相生为吉
- 男命纳音生女命纳音：男爱女，男助女，吉。
- 女命纳音生男命纳音：女爱男，女助男，吉。
- 男女纳音相同：比和，平。
- 男女纳音相生相成：大吉，恩爱长久。

### 2. 纳音相克为忌
- 男命纳音克女命纳音：男压女，婚姻有压力，次吉。
- 女命纳音克男命纳音：女压男，婚姻不顺，忌。
- 男女纳音天克地冲：大忌，婚姻难成或离散。

### 3. 纳音合婚口诀
- 天河水不惧任何土（天上水不入土制）
- 大海水不惧任何土（海大土难制）
- 大林木不惧任何金（林大金伐不尽）
- 杨柳木怕剑锋金（柳柔易折）
- 霹雳火怕天上水（雷霆火逢天水灭）
- 海中金不怕火（海深火难及）

### 4. 纳音合婚实战要点
- 纳音合婚为古法，仅作参考，非决定婚姻成败之核心。
- 婚姻成败核心看正五行、十神（配偶星、配偶宫）、用神喜忌（详见 11_婚恋关系规则卡.md）。
- 纳音合婚与现代正五行合婚冲突时，以正五行十神为主。
- 古人纳音合婚多配对八字格局综合判断，非单看纳音。

## 四、纳音格局与强弱

### 1. 纳音强弱
纳音五行亦有旺衰强弱，看其是否当令、有根：
- 纳音当令（如纳音为火，生于巳午月）：强。
- 纳音休囚（如纳音为火，生于亥子月）：弱。
- 纳音受克（如纳音为木，原局金多克之）：弱。
- 纳音受生（如纳音为木，原局水多生之）：强。

### 2. 纳音格局
- **纳音三合**：年月日时纳音三合一方，主该五行旺势。
- **纳音相生**：四柱纳音顺生，主流通有情，主一生顺遂。
- **纳音相克**：四柱纳音相战，主流通受阻，主一生波折。
- **纳音相同**：四柱纳音多同，主该五行偏旺。

### 3. 纳音断性情
- 金命人：刚毅果决、重义气。
- 木命人：仁慈正直、上进。
- 水命人：聪明灵活、善谋。
- 火命人：热情外向、急躁。
- 土命人：忠厚稳重、守信。

具体纳音再细分（如海中金沉静、剑锋金刚烈）。

## 五、纳音与正五行主次使用边界

**核心原则：正五行、十神、格局、用神为核心；纳音、神煞仅作辅助参考。**

### 1. 纳音为辅
- 纳音力量较弱，论命以正五行（天干地支本气）为根本。
- 纳音主要用于：
  1. 看年命（年柱纳音），又称"命主"，定性情基调。
  2. 合婚参考（年命纳音相生克）。
  3. 性情推断（纳音意象）。
  4. 择日、民间俗法。

### 2. 正五行为主
- 格局成败、用神取用、十神论事、岁运吉凶，皆以正五行为准。
- 纳音与正五行结论冲突时，以正五行为准。
- 纳音仅作正五行之外的补充印证。

### 3. 使用边界
- 正五行定大格局、富贵贫贱层次。
- 纳音辅助定性情、年命、合婚参考。
- 神煞辅助点睛（如驿马、桃花）。
- 三者主次分明，不可本末倒置。

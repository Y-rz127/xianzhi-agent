# 方位迁移断法

> 八字看方位吉凶、外出发展、迁移动迁、驿马动象、海外发展的专项断法规则。供方位迁移 Worker 检索引用。

## 一、总则

1. **用神定吉方**：用神五行对应方位，宜向用神方位发展、避开忌神方位。
2. **驿马定动象**：驿马星主奔波动迁，逢冲更明显。
3. **大运流年引动**：走驿马运、外地运，或流年冲动日支/驿马，多为外出之期。
4. **本地 vs 外地**：日主强弱、用神所在、驿马旺衰共同决定宜守宜动。

## 二、用神方位

### 1. 五行方位对应

| 五行 | 方位 | 卦 |
|------|------|----|
| 木 | 东方 | 震 |
| 火 | 南方 | 离 |
| 金 | 西方 | 兑 |
| 水 | 北方 | 坎 |
| 土 | 中央/本地 | 中 |

### 2. 按用神定吉方

- **用神为木**：宜东方发展、求学创业。
- **用神为火**：宜南方发展、成名立业。
- **用神为金**：宜西方发展、武职金融。
- **用神为水**：宜北方发展、流通航运。
- **用神为土**：宜本地或中原地区、地产农业。

### 3. 忌神方位

- 忌神之方位宜避：如忌火则不宜久居南方。
- 但忌神被制化时，方位凶性减弱。

### 4. 方位与城市对应参考

| 方位 | 代表城市/地区 |
|------|---------------|
| 东方 | 上海、杭州、南京、苏州、青岛、厦门 |
| 南方 | 广州、深圳、珠海、海口、南宁、长沙 |
| 西方 | 成都、重庆、西安、昆明、贵阳、兰州 |
| 北方 | 北京、天津、大连、哈尔滨、沈阳、济南 |
| 中央 | 武汉、郑州、合肥、长沙（偏南） |

### 5. 方位判断注意事项

- 方位以出生地为原点判断，非绝对地理方位。
- 同一省内不同城市，方位差异不大，可忽略。
- 跨省迁移才需重点看方位吉凶。
- 海外发展需结合驿马和用神综合判断。

## 三、驿马星

### 1. 查法（年支或日支三合局长生对冲）

| 三合局 | 驿马 |
|--------|------|
| 申子辰 | 寅 |
| 寅午戌 | 申 |
| 巳酉丑 | 亥 |
| 亥卯未 | 巳 |

### 2. 驿马作用

- **驿马逢冲**：动象最明显，主远行、迁居、调动。
- **驿马逢合**：动象被绊，欲动不能。
- **驿马坐财官**：因财官而动、动中得财官。
- **驿马坐忌神**：奔波劳碌、外出辛苦。
- **驿马坐空亡**：动象虚浮、有外出之心无外出之实。

### 3. 驿马与职业

- 驿马坐财：流动求财、贸易、运输、出差多的工作。
- 驿马坐官：因公出差、调动频繁、外派任职。
- 驿马坐印：因学业外出、学术交流、异地求学。
- 驿马坐食伤：以技艺流动、演出、巡回、自媒体。

### 4. 双驿马/多驿马

- 年支驿马+日支驿马：一生多迁移、居无定所。
- 驿马逢冲+驿马逢冲：动上加动、奔波不停。
- 多驿马为用：适合频繁出差、跨国业务。
- 多驿马为忌：奔波劳碌、居无定所、家庭不稳。

## 四、宫位与迁移

### 1. 年柱看祖籍
- 年柱逢冲：早年离祖、不在出生地发展。
- 年柱驿马：祖上多迁移、自身也动。
- 年柱印星为用：祖业可守、宜在出生地发展。

### 2. 日支看自身动向
- 日支逢冲：自身或配偶有动象。
- 日支驿马：中年外出多。
- 日支合他柱：被绊住、动不了。

### 3. 时柱看晚年
- 时柱驿马：晚年仍有动象。
- 时柱逢冲：晚境变动。
- 时柱印星为用：晚年安居。

## 五、大运流年引动

- **走驿马运**：十年内多奔波动迁。
- **流年冲动日支/驿马**：当年有外出、调动、搬迁之象。
- **走外地运**（年柱被冲、驿马被冲）：主外出发展。
- **太岁冲日支**：当年居家或工作变动。
- **流年驿马逢合**：欲动不能、暂缓。
- **流年驿马逢冲**：强动信号、必动之年。

## 六、本地 vs 外地

### 1. 宜外出
- 日主强、能任财官：宜外出求发展。
- 驿马旺、用神在他方：宜外出。
- 年柱逢冲、祖业难守：宜离乡。
- 比劫为忌、本地竞争大：宜外出避争。
- 食伤生财为用、外地机会多：宜外出。

### 2. 宜守本地
- 日主弱、用神在本地：宜守。
- 印星为用、得父母荫蔽：宜守本地。
- 驿马被合、动象被绊：宜暂守。
- 财星在本地得力：本地求财更稳。
- 官印相生、本地有平台：宜守本地发展。

### 3. 先守后出
- 早年走印比运（强身）、中年走食伤财运（求财）：先在本地积累，后外出发展。
- 日主偏弱需先强身，再外出求财。

### 4. 先出后归
- 早年走驿马运、中年走印比运：早年外出、中年归乡。
- 年柱逢冲、时柱印星为用：早年离乡、晚年归故里。

## 七、海外发展判断

### 1. 适合海外发展
- 驿马旺且逢冲：动象强烈、适合远行。
- 用神在远方（忌神在本地）：海外发展更吉。
- 偏财为用通根：适合跨国贸易、海外投资。
- 日主强、食伤旺：有闯劲、适应力强。

### 2. 不适合海外发展
- 印星为用、根在本地：离乡则失靠山。
- 日主弱、驿马为忌：外出辛苦、难以适应。
- 比劫为用、团队在本地：离队则势单力薄。
- 官杀为用、平台在本地：离平台则失势。

### 3. 海外发展应期
- 大运走驿马旺地 + 流年冲动日支 → 出国之期。
- 大运走偏财运 + 流年驿马逢冲 → 海外求财之期。
- 流年驿马逢冲 + 偏财透干 → 海外机会出现之年。

## 八、搬迁后运势变化

### 1. 搬迁到用神方位
- 运势提升、机会增多、贵人出现。
- 事业财运好转、身体状态改善。
- 人际关系更顺畅。

### 2. 搬迁到忌神方位
- 运势下滑、阻碍增多、小人出现。
- 事业财运受阻、身体状态变差。
- 人际关系紧张。

### 3. 搬迁过渡期
- 搬迁后1-2年为适应期，运势波动属正常。
- 用神方位：适应期短、很快好转。
- 忌神方位：适应期长、持续不顺。
- 中性方位：平稳过渡、无大变化。

## 九、迁移应期参考

1. 大运走驿马 + 流年冲动日支 → 强外出信号。
2. 大运冲年柱 + 流年驿马 → 离祖之期。
3. 流年透财官 + 驿马动 → 因财官而动。
4. 流年合驿马 → 欲动不能、暂缓。
5. 大运走驿马 + 流年驿马逢冲 → 必动之年。
6. 太岁冲年柱 + 流年驿马 → 离乡之期。

## 十、合规口径

1. 迁移方位仅为命理趋势参考，实际决策需结合工作、家庭、政策等现实条件。
2. 不绝对断"必须去某方""某方大凶"，应说明依据与变数。
3. 涉及重大迁移决策，劝导综合现实因素，不夸大命理作用。
4. 海外发展涉及签证、法律、文化适应等，劝导以专业咨询为准。

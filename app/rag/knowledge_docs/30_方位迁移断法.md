# 方位迁移断法

> 八字看方位吉凶、外出发展、迁移动迁、驿马动象的专项断法规则。供方位迁移 Worker 检索引用。

## 一、总则

1. **用神定吉方**：用神五行对应方位，宜向用神方位发展、避开忌神方位。
2. **驿马定动象**：驿马星主奔波动迁，逢冲更明显。
3. **大运流年引动**：走驿马运、外地运，或流年冲动日支/驿马，多为外出之期。
4. **本地 vs 外地**：日主强弱、用神所在、驿马旺衰共同决定宜守宜动。

## 二、用神方位

### 1. 五行方位对应

| 五行 | 方位 | 卦 |
|------|------|----|
| 木 | 东方 | 震 |
| 火 | 南方 | 离 |
| 金 | 西方 | 兑 |
| 水 | 北方 | 坎 |
| 土 | 中央/本地 | 中 |

### 2. 按用神定吉方

- **用神为木**：宜东方发展、求学创业。
- **用神为火**：宜南方发展、成名立业。
- **用神为金**：宜西方发展、武职金融。
- **用神为水**：宜北方发展、流通航运。
- **用神为土**：宜本地或中原地区、地产农业。

### 3. 忌神方位

- 忌神之方位宜避：如忌火则不宜久居南方。
- 但忌神被制化时，方位凶性减弱。

## 三、驿马星

### 1. 查法（年支或日支三合局长生对冲）

| 三合局 | 驿马 |
|--------|------|
| 申子辰 | 寅 |
| 寅午戌 | 申 |
| 巳酉丑 | 亥 |
| 亥卯未 | 巳 |

### 2. 驿马作用

- **驿马逢冲**：动象最明显，主远行、迁居、调动。
- **驿马逢合**：动象被绊，欲动不能。
- **驿马坐财官**：因财官而动、动中得财官。
- **驿马坐忌神**：奔波劳碌、外出辛苦。

## 四、宫位与迁移

### 1. 年柱看祖籍
- 年柱逢冲：早年离祖、不在出生地发展。
- 年柱驿马：祖上多迁移、自身也动。

### 2. 日支看自身动向
- 日支逢冲：自身或配偶有动象。
- 日支驿马：中年外出多。

### 3. 时柱看晚年
- 时柱驿马：晚年仍有动象。
- 时柱逢冲：晚境变动。

## 五、大运流年引动

- **走驿马运**：十年内多奔波动迁。
- **流年冲动日支/驿马**：当年有外出、调动、搬迁之象。
- **走外地运**（年柱被冲、驿马被冲）：主外出发展。
- **太岁冲日支**：当年居家或工作变动。

## 六、本地 vs 外地

### 1. 宜外出
- 日主强、能任财官：宜外出求发展。
- 驿马旺、用神在他方：宜外出。
- 年柱逢冲、祖业难守：宜离乡。

### 2. 宜守本地
- 日主弱、用神在本地：宜守。
- 印星为用、得父母荫蔽：宜守本地。
- 驿马被合、动象被绊：宜暂守。

## 七、迁移应期参考

1. 大运走驿马 + 流年冲动日支 → 强外出信号。
2. 大运冲年柱 + 流年驿马 → 离祖之期。
3. 流年透财官 + 驿马动 → 因财官而动。
4. 流年合驿马 → 欲动不能、暂缓。

## 八、合规口径

1. 迁移方位仅为命理趋势参考，实际决策需结合工作、家庭、政策等现实条件。
2. 不绝对断"必须去某方""某方大凶"，应说明依据与变数。
3. 涉及重大迁移决策，劝导综合现实因素，不夸大命理作用。

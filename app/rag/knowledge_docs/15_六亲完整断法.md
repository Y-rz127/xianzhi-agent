# 六亲完整断法

> 六亲为父母、兄弟、配偶、子女、祖辈之缘。本篇含六亲取用、宫位、吉凶、助力拖累、应期、流年细化。
> 引用典籍：《渊海子平·论六亲》《三命通会·论六亲》《滴天髓·论六亲》。
> **核心原则：六亲看"星+宫"配合，星为缘、宫为位，星宫同看方准。**

## 一、六亲取用表

六亲取用以十神为本，男女有别。

### 1. 男命六亲取用

| 六亲 | 十神 | 说明 |
|------|------|------|
| 祖父 | 偏财 | 偏财为祖父（偏财为父之父） |
| 祖母 | 正印 | 正印为祖母（生父者为祖母） |
| 父亲 | 偏财 | 偏财为父（克印者为父） |
| 母亲 | 正印 | 正印为母（生身者为母） |
| 兄弟 | 比肩 | 同我者为兄弟 |
| 姐妹 | 劫财 | 同我异阴者为姐妹 |
| 妻子 | 正财 | 我克者为妻，正财为正妻 |
| 情人/继室 | 偏财 | 偏财为偏房、情人 |
| 儿子 | 七杀 | 克日干者（书房派取食伤为子，以食伤为主） |
| 女儿 | 正官 | 克日干者（书房派取食伤为女） |

> 注：男命子女取用，书房派多取食伤（我生者为子女），盲派有取官杀者。实战以食伤为主、官杀为辅参考。

### 2. 女命六亲取用

| 六亲 | 十神 | 说明 |
|------|------|------|
| 祖父 | 正财 | 正财为祖父 |
| 祖母 | 偏印 | 偏印为祖母 |
| 父亲 | 正财 | 正财为父（女命以财为父） |
| 母亲 | 偏印 | 偏印为母（亦有取正印者） |
| 兄弟 | 比肩 | 同 |
| 姐妹 | 劫财 | 同 |
| 丈夫 | 正官 | 克我者为夫，正官为正夫 |
| 情人/继夫 | 七杀 | 七杀为偏夫、情人 |
| 儿子 | 伤官 | 我生者为子，女命以伤官为子 |
| 女儿 | 食神 | 食神为女 |

> 注：女命六亲取用流派分歧较大。主流以正官为夫、七杀为偏夫、食伤为子女、偏印为母、正财为父。

## 二、六亲宫位

四柱宫位对应六亲位置：

| 宫位 | 年龄段 | 六亲 | 人生阶段 |
|------|--------|------|----------|
| 年柱 | 1-16岁（幼少年） | 祖父母、祖上、长辈 | 祖荫、幼年环境 |
| 月柱 | 16-32岁（青年） | 父母、兄弟 | 家庭出身、青年运势 |
| 日柱 | 32-48岁（中年） | 配偶（日支）、自身 | 婚姻、中年事业 |
| 时柱 | 48岁后（晚年） | 子女、晚年 | 子息、晚年运势 |

- **年柱**：祖辈、祖业、幼年家境、远方。
- **月柱**：父母、兄弟、家庭出身、青年。
- **日支**：配偶、夫妻关系、中年。
- **时柱**：子女、晚年、下属、归宿。

## 三、父母断法

### 1. 父亲
- **取星**：男命偏财为父，女命正财为父。
- **看宫**：月柱为父母宫。
- **父亲得力**：父星为喜用且通根透干、月柱生扶日主、父星无伤。
- **父亲缘薄**：父星为忌、父星被冲克、父星入墓逢空、父星逢比劫夺。
- **父亲健康**：父星被冲克岁运、父星入墓逢冲开库、父星逢羊刃七杀。
- **父亲早逝**：父星逢空亡、父星被克尽、月柱逢冲克严重。

### 2. 母亲
- **取星**：男命正印为母，女命偏印为母。
- **看宫**：月柱为父母宫。
- **母亲得力**：母星为喜用且通根透干、月柱生扶日主、母星无伤。
- **母亲缘薄**：母星为忌、母星被冲克、母星入墓逢空、财坏印（财克印，母受伤）。
- **母亲健康**：母星被冲克岁运、母星入墓逢冲开库、母星逢羊刃七杀。
- **母亲早逝**：母星逢空亡、母星被财克尽、月柱逢冲克严重。
- **父母关系**：财坏印（父克母）、父星与母星相冲、月柱刑冲。

## 四、兄弟姐妹断法

- **取星**：比肩为兄弟，劫财为姐妹。
- **看宫**：月柱为兄弟宫。
- **兄弟多而得力**：比劫旺为喜用、月柱比劫透干通根、比劫生扶日主。
- **兄弟多而争财**：比劫旺为忌、比劫夺财、兄弟争产、合伙内讧。
- **兄弟缘薄**：比劫逢空亡、比劫被冲克、比劫入墓、月柱逢冲。
- **兄弟健康**：比劫被冲克岁运、比劫逢羊刃七杀。

## 五、配偶断法

### 1. 配偶星
- **男命**：正财为妻，偏财为情人/继室。
- **女命**：正官为夫，七杀为情人/继夫。

### 2. 配偶宫
- **日支**：配偶宫。

### 3. 配偶得力
- 配偶星为喜用且通根透干、日支坐喜用神、配偶星与日主相生相合。
- 配偶星逢生扶岁运、配偶星从墓库中透出。

### 4. 配偶缘薄
- 配偶星为忌、配偶星被冲克、配偶星入墓逢空。
- 日支逢冲刑害、日支坐忌神。
- 比劫夺财（男）、伤官见官（女）。

### 5. 配偶健康
- 配偶星被冲克岁运、配偶星入墓逢冲开库。
- 日支逢冲刑、日支逢羊刃七杀。

### 6. 配偶外遇
- 配偶星被合走（如男命财星被合、女命官星被合）。
- 墙外桃花逢引动、墙外桃花+配偶星被合。
- 正偏财混杂（男）/正偏官混杂（女）且无制。

## 六、子女断法

### 1. 子女星
- **男命**：七杀为子（书房派取食伤为子女）。
- **女命**：伤官为子，食神为女。

### 2. 子女宫
- **时柱**：子女宫。

### 3. 子女得力
- 子女星为喜用且通根透干、时柱生扶日主、时柱坐喜用神。
- 子女星逢生扶岁运、时柱逢生合。

### 4. 子女缘薄
- 子女星为忌、子女星被冲克、子女星入墓逢空。
- 时柱逢冲刑害、时柱坐忌神。
- 食伤逢枭夺（女命枭神夺食，子女受损）。

### 5. 子女数量
- 子女星多而旺（且为喜用）：子女多。
- 子女星少而弱（或逢空亡）：子女少或无。
- 子女星逢冲克：子女有损。
- 时柱逢冲：子女缘薄或子女远离。

### 6. 子女性别
- 书房派：男命食神为女、伤官为子；女命食神为女、伤官为子。
- 盲派：男命七杀为子、正官为女；女命伤官为子、食神为女。
- 时柱阳干阳支多生男，阴干阴支多生女（参考）。

## 七、祖辈断法

- **取星**：男命偏财为祖父、正印为祖母；女命正财为祖父、偏印为祖母。
- **看宫**：年柱为祖辈宫。
- **祖业丰厚**：年柱生扶日主、年柱坐喜用神、祖辈星为喜用。
- **祖业破败**：年柱逢冲克、年柱坐忌神、祖辈星被克尽。
- **祖辈缘薄**：年柱逢空亡、祖辈星逢空亡、年柱逢冲。

## 八、六亲吉凶判断要点

### 1. 六亲星为用神且无伤
- 该六亲得力、有助、缘深、健康。
- 主该六亲为贵人，扶助日主发展。

### 2. 六亲星为忌神且旺
- 该六亲拖累、缘薄、多灾、压制日主。
- 主该六亲为负担，牵累日主。

### 3. 六亲星被冲克合化
- 该六亲缘薄、分离、灾病、变化。
- 冲克重则灾重，冲克轻则波动。

### 4. 六亲宫被冲克
- 该宫位所主六亲不安。
- 如日支被冲，配偶不稳；时柱被冲，子女不安。

### 5. 六亲星入墓逢空
- 该六亲缘薄、早逝、分离、无依。
- 入墓：该六亲病、灾、困。
- 逢空：该六亲虚缘、分离、无实质。

## 九、六亲应期

### 1. 六亲灾应期
- 六亲星被冲克岁运：该六亲当年有灾。
- 六亲星入墓逢冲开库：该六亲当年有大灾。
- 六亲宫被冲克岁运：该宫位六亲当年有变化（灾、分离、病）。
- 六亲星逢羊刃七杀岁运：该六亲当年有刑伤。

### 2. 六亲助益应期
- 六亲星逢生合岁运：该六亲当年助益日主。
- 六亲宫逢生合岁运：该宫位六亲当年有喜。
- 六亲星从墓库中透出：该六亲当年显力。

### 3. 六亲分离应期
- 六亲星被合走：该六亲与他人合，分离日主。
- 六亲宫被冲：该宫位六亲分离、变动。
- 六亲星逢空亡：该六亲虚缘、分离。

## 十、六亲与流年细化

### 1. 父母流年
- 流年冲月柱：父母事、家庭变动。
- 流年克父星：父亲灾。
- 流年克母星：母亲灾。
- 流年生扶父星：父亲有喜。
- 流年生扶母星：母亲有喜。

### 2. 配偶流年
- 流年冲日支：配偶事、婚姻变动。
- 流年合日支：婚姻动（结婚或外遇）。
- 流年克配偶星：配偶灾。
- 流年生扶配偶星：配偶有喜。
- 流年引动桃花+配偶星同动：婚恋应期。

### 3. 子女流年
- 流年冲时柱：子女事、子女变动。
- 流年克子女星：子女灾。
- 流年生扶子女星：子女有喜。
- 流年引动子女星+时柱同动：添丁应期。

### 4. 兄弟流年
- 流年冲月柱：兄弟事。
- 流年克比劫：兄弟灾。
- 流年生扶比劫：兄弟有喜。
- 流年比劫夺财：兄弟争产、合伙内讧。

## 十一、六亲与格局配合

- **官印相生格**：父母有地位、有文化、得父母力。
- **财官双美格**：配偶得力、夫妻和睦、事业有成。
- **伤官见官格**：婚姻不稳、与上级冲突、子女有损（女命）。
- **比劫夺财格**：兄弟争财、配偶被夺、破财。
- **枭神夺食格**：子女受损（女命）、财源断、福薄。
- **食神生财格**：子女有出息（女命）、经营有道、夫妻和睦。
- **印旺身弱格**：母亲过度保护、配偶受压制、依赖长辈。

## 十二、回答口径

- 先说六亲缘：缘深缘薄？得力拖累？
- 再说六亲灾：何时有灾？何六亲？
- 最后说建议：如何化解？如何相处？
- 引经据典佐证，结论先给白话，再引古籍原文。
- 不说"必克""必离"，要给条件："若……则……"。

# 合冲刑害规则卡

> 合冲刑害是地支互动的核心，决定人事聚散冲突。本篇含力量排序、三刑细分、六害、贪合忘冲判定。
> 引用典籍：《渊海子平·论地支》《三命通会·论合冲刑害》。

## 一、合局力量排序

合局力量由强到弱，决定合化成败：

**三会局 ＞ 三合局 ＞ 六合 ＞ 半合 ＞ 拱合**

### 1. 三会局（方合，最强）
- 寅卯辰三会东方木局
- 巳午未三会南方火局
- 申酉戌三会西方金局
- 亥子丑三会北方水局

三支会一方之气，力量最猛，成则一方专旺。多成专旺、从强之格。

### 2. 三合局（次强）
- 申子辰三合水局
- 亥卯未三合木局
- 寅午戌三合火局
- 巳酉丑三合金局

三支合一方之气，成则化神党势。

### 3. 六合（中）
- 子丑合化土
- 寅亥合化木
- 卯戌合化火
- 辰酉合化金
- 巳申合化水
- 午未合化土

两支合化，力量中等。多主牵绊、和合。

### 4. 半合（弱）
三合仅见二支：旺半合（子辰、卯未、午戌、酉丑）、生半合（申子、亥卯、寅午、巳酉）。力量次于全合。

### 5. 拱合（最弱）
三合见首尾两支，中间虚一（如申辰拱子），待岁运填实方合。

## 二、六冲

### 1. 六冲组合
- 子午冲（水火冲）
- 卯酉冲（木金冲）
- 寅申冲（木金冲）
- 巳亥冲（火水冲）
- 辰戌冲（土土冲）
- 丑未冲（土土冲）

### 2. 六冲轻重
- **近冲重**：相冲两支紧贴（如日支冲月支、日支冲时支）力量大，主明显变动。
- **远冲轻**：相冲两支远隔（如年支冲时支）力量小，主暗动。
- **冲喜神吉**：冲去忌神则吉（如忌神被冲去，反主福）。
- **冲忌神凶**：冲去用神则凶（如用神被冲，主灾）。
- **冲库开库**：辰戌冲、丑未冲为冲土库，开库则库中所藏透出，主财官出库。

### 3. 六冲宫位分断
- 冲年柱：环境、家族、外部背景变动。
- 冲月柱：事业平台、父母、工作结构变动。
- 冲日支：婚姻、居住、配偶变动（婚姻不稳之兆）。
- 冲时柱：子女、晚年、长期规划变动。

### 4. 六冲主事
- 子午冲：水火相战，主心肾、血压、情绪动荡。
- 卯酉冲：木金相战，主肝肺、交通、背井离乡。
- 寅申冲：木金相战，主奔波、肝胆、迁移。
- 巳亥冲：水火相战，主心肾、婚姻、远行。
- 辰戌冲：土冲开库，主房产、破财、官非。
- 丑未冲：土冲开库，主房产、刑伤、六亲。

## 三、三刑（细分）

三刑主官非、刑伤、内耗、压力。分三类：

### 1. 寅巳申三刑（无恩之刑）
- **组合**：寅刑巳、巳刑申、申刑寅。
- **性质**：无恩刑，主恩将仇报、忘恩负义、知恩不报反为仇。
- **应事**：官非口舌、交通事故、刑伤、人际背叛。
- **现实**：施恩反遭怨、好心办坏事、被信任之人出卖。

### 2. 丑未戌三刑（恃势之刑）
- **组合**：丑刑戌、戌刑未、未刑丑。
- **性质**：恃势刑，主仗势欺人、强梁专横、以势压人。
- **应事**：官非、刑伤、破财、六亲刑克、土地房产纠纷。
- **现实**：恃强凌弱招祸、权势斗争、官讼是非。

### 3. 辰午酉亥自刑
- **组合**：辰刑辰、午刑午、酉刑酉、亥刑亥（同支自见）。
- **性质**：自刑，主自我惩罚、自寻烦恼、内耗、重复犯错。
- **应事**：内耗焦虑、自残、抑郁、重复犯同类错误、自我封闭。
- **现实**：钻牛角尖、自我折磨、明知故犯。

### 4. 三刑轻重与化解
- 三支齐现（如寅巳申全）为真刑，力量大，主大灾。
- 两支见（如寅巳、巳申、申寅）为半刑，力量次之，仍主刑伤。
- 刑逢冲合则减力：刑被冲破、刑被合解。
- 刑为忌神旺则大凶，刑为喜用（少见）反主威权。

## 四、六害

六害偏隐性，不如冲明显，主暗耗、隔阂、不顺。

### 1. 六害组合
- 子未害
- 丑午害
- 寅巳害
- 卯辰害
- 申亥害
- 酉戌害

### 2. 六害细分应事
- **子未害**：水害土，主骨肉、六亲隔阂，父母缘薄。
- **丑午害**：土害火，主心力交瘁、内耗、性情不和。
- **寅巳害**：木害火，主口舌是非、人际背叛、外伤。
- **卯辰害**：木害土，主手足、邻里不和、房产纠纷。
- **申亥害**：金害水，主肾、泌尿、夫妻隐疾。
- **酉戌害**：金害土，主肺、皮肤、口舌暗耗。

### 3. 六害主事
- 感情：隐性隔阂、沟通不畅、貌合神离。
- 事业：流程阻力、暗线阻力、合作不舒服。
- 六亲：缘薄、暗中损耗。

## 五、合冲刑害同时出现：贪合忘冲、贪冲忘合

原局或岁运同时见合与冲、冲与刑时，须判断何者先发生。

### 1. 贪合忘冲
- 原局同时见合与冲，**合者先合、忘其冲**。
- 例：子午冲，若子又与丑合，则子贪合丑而忘冲午，冲力减。
- 判定：合紧贴、合化条件足者，先合忘冲。

### 2. 贪冲忘合
- 原局同时见冲与合，**冲者先冲、忘其合**。
- 例：子丑合，若子又被午冲，冲力大于合力，则子被冲而忘合丑。
- 判定：冲紧贴、冲力大者，先冲忘合。

### 3. 合冲力量比较
- 三会、三合之力 ＞ 冲之力（合能解冲）。
- 六合之力 ≈ 冲之力（须看紧贴远近、化神当令否）。
- 半合、拱合之力 ＜ 冲之力（冲能破合）。

### 4. 刑冲合害并存
- 刑冲同见：以冲为主，刑为辅（冲力大于刑力）。
- 合刑同见：合能解刑（贪合忘刑）。
- 害与冲：冲为主、害为辅。

## 六、使用原则

1. **合冲刑害不能孤立断事**，须落到"哪个柱、哪个宫位、哪个十神、哪个大运流年触发"。
2. **看力量**：紧贴者力大，远隔者力小；化神当令者力大。
3. **看喜忌**：合冲刑害喜神则吉，冲刑忌神则吉。
4. **翻译成现实语言**：变化、合作、压力、人际、财务、关系稳定性，不堆术语。
5. **宫位+十神+神煞综合**：冲日支+伤官+羊刃，主婚姻大灾；冲时柱+七杀+空亡，主子女灾。

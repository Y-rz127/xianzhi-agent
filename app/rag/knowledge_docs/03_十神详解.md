# 十神详解

> 十神是八字论命的核心语言，由日主与其他天干地支的五行生克关系得出。本篇涵盖十神性情、宫位分断、组合吉凶、六亲取用、清浊真假。
> 引用典籍：《渊海子平·论十神》《子平真诠·论用神成败》。

## 一、十神定义与分类

十神由日主（日柱天干）与其他天干地支的五行生克关系得出。

| 十神 | 关系 | 阴阳 | 性情 | 主事 |
|------|------|------|------|------|
| 比肩 | 同我 | 同阴阳 | 独立、竞争、自尊 | 兄弟朋友、同辈、合作 |
| 劫财 | 同我 | 异阴阳 | 争夺、冲动、损耗 | 兄弟、竞争者、破财 |
| 食神 | 我生 | 同阴阳 | 才华、福寿、温和 | 子女、口福、技艺 |
| 伤官 | 我生 | 异阴阳 | 聪明、叛逆、口才 | 才艺、官非、子息 |
| 偏财 | 我克 | 同阴阳 | 横财、慷慨、父亲 | 偏业、意外财、情人 |
| 正财 | 我克 | 异阴阳 | 正业、节俭、妻子 | 正业收入、妻、稳定财 |
| 正官 | 克我 | 异阴阳 | 事业、守法、地位 | 官职、丈夫（女）、名誉 |
| 七杀 | 克我 | 同阴阳 | 权威、压力、险厄 | 武职、偏官、灾厄 |
| 正印 | 生我 | 异阴阳 | 学业、仁慈、母亲 | 学历、母亲、庇护 |
| 偏印 | 生我 | 同阴阳 | 孤独、奇才、继母 | 偏学、继母、孤僻 |

### 1. 十神吉凶分类
- **吉神（四善神）**：正官、正印、正财、食神。主平稳、顺用（生扶保护）。
- **凶神（四恶神）**：七杀、偏印、劫财、伤官。主变动、逆用（制化克泄）。
- 比肩中性，可吉可凶，看旺衰。

### 2. 顺用与逆用（《子平真诠》核心）
- **善神顺用**：吉神宜生扶保护。如正官喜财生、印护，忌伤官见官、七杀混。
- **恶神逆用**：凶神宜制化克泄。如七杀宜食制、印化，忌官杀混杂无制。
- 顺逆得宜则格局成，逆顺反错则格局败。

## 二、十神在四柱宫位分断

四柱宫位代表人生不同阶段与六亲长辈，十神落何宫位，主事不同。

| 宫位 | 年龄段 | 六亲 | 人生阶段 |
|------|--------|------|----------|
| 年柱 | 1-16岁（幼少年） | 祖父母、祖上、长辈 | 祖荫、幼年环境 |
| 月柱 | 16-32岁（青年） | 父母、兄弟 | 家庭出身、青年运势 |
| 日柱 | 32-48岁（中年） | 配偶（日支）、自身 | 婚姻、中年事业 |
| 时柱 | 48岁后（晚年） | 子女、晚辈 | 晚年、子息、归宿 |

### 1. 年柱十神分断
- **年柱正官**：祖上有官贵，幼年得长辈庇护，出身好。
- **年柱七杀**：祖上多武职或破败，幼年环境动荡，出身艰辛。
- **年柱正印**：祖上有学问，幼年得母亲祖辈疼爱，书香门第。
- **年柱偏印**：祖上偏业，幼年与母亲缘薄，或离祖过继。
- **年柱正财**：祖上富裕，幼年家境好。
- **年柱偏财**：祖上经商，幼年家境起伏，父亲能干。
- **年柱食神**：祖上有福寿，幼年安康。
- **年柱伤官**：祖上易破败，幼年顽皮叛逆，与长辈缘薄。
- **年柱比劫**：祖上平平，幼年兄弟多，家境一般。

### 2. 月柱十神分断（月令为格局之根，最重）
- **月柱正官**：父母有地位，青年得父母助力，事业早成。
- **月柱七杀**：父母刚强或压力，青年多磨练，自立创业。
- **月柱正印**：父母慈祥有文化，青年学业顺，得长辈提携。
- **月柱偏印**：父母偏业或缘薄，青年孤僻好学，奇才。
- **月柱正财**：父母勤俭持家，青年家境稳，事业稳进。
- **月柱偏财**：父亲能干善经营，青年家境波动，善理财。
- **月柱食神**：父母温和有福，青年多才多艺，事业顺。
- **月柱伤官**：父母聪明但易冲突，青年叛逆聪明，才艺早显。
- **月柱比劫**：兄弟多，青年竞争大，自立或合伙。

### 3. 日柱十神分断（日支为配偶宫，看婚姻中年）
- **日支正官**：配偶端正有责任感，婚姻稳，中年事业顺。
- **日支七杀**：配偶刚强有魄力，婚姻多磨，配偶能干但压人。
- **日支正印**：配偶贤淑包容，婚姻温馨，得配偶之助。
- **日支偏印**：配偶孤僻多疑，婚姻冷淡，沟通难。
- **日支正财**：配偶勤俭持家，婚姻稳，得妻助致富。
- **日支偏财**：配偶慷慨外向，婚姻多情波动，妻能干。
- **日支食神**：配偶温和有才，婚姻幸福，子女有出息。
- **日支伤官**：配偶聪明任性，婚姻易冲突，克配偶。
- **日支比劫**：配偶同辈如友，婚姻易有第三者，分财。

### 4. 时柱十神分断（时柱为子女宫，看晚年子息）
- **时柱正官**：子女端正有出息，晚年得子女奉养，安乐。
- **时柱七杀**：子女刚强难管，晚年多操心，子女有作为但叛逆。
- **时柱正印**：子女孝顺有学问，晚年安享，子女庇护。
- **时柱偏印**：子女孤僻，晚年孤独，与子女缘薄。
- **时柱正财**：子女勤俭致富，晚年丰足，子女奉养。
- **时柱偏财**：子女经商有道，晚年富裕，但子女在外。
- **时柱食神**：子女有福有才，晚年享福，子女孝顺。
- **时柱伤官**：子女聪明任性，晚年操心，子女离祖。
- **时柱比劫**：子女多竞争，晚年劳碌，财产被分。

## 三、十神组合吉凶（经典组合）

### 1. 吉组合
- **官印相生**：正官生正印，印生身。主事业顺、有学历、有地位、得名声。文人、公职之贵格。
- **财官双美**：财生官、官卫财。主富贵双全，事业财源两旺。
- **食神生财**：食神泄秀生财。主以技艺才华致富，商人之格。
- **食神制杀**：食神制七杀。主以才制权，武职、将帅之格，富贵有威。
- **杀印相生**：七杀生偏印、印生身。主掌大权，威权显赫，将相之格。
- **财佩印**：财印不碍，财养命、印护身。主富贵有学，文人富格。
- **官星佩印**：正官配正印，主清贵有德，仕途顺遂。

### 2. 凶组合
- **伤官见官**：伤官克正官。主口舌官非、与领导冲突、婚姻不稳（女命尤忌）。《子平真诠》云："伤官见官，为祸百端。"
- **官杀混杂**：正官七杀同现。主事多反复、婚姻不稳（女命尤忌）、是非不断。
- **比劫夺财**：比劫多克正偏财。主破财、夺妻（男命）、兄弟争产。
- **枭神夺食**：偏印克食神。主断财源、失福、子息受损、灾病。
- **财多身弱**：财旺身弱不能任。主富屋贫人，财多压身，因财生灾。
- **官多变杀**：官多不贵反作杀论，主压力大、是非多。
- **杀重身轻**：七杀重身弱无制。主灾病、压力、易遭官非横祸。
- **群比争财**：比劫成群夺财。主破财克妻、一生劳碌。
- **印旺身衰**（母多灭子）：印过旺反克身（金多水浊类）。主懒惰、无成、依赖。

### 3. 十神生克链（论事顺序）
- 官生印，印生身，身生食伤，食伤生财，财生官。循环则流通有情，主一生顺遂。
- 任何一环被克断，则流通受阻，对应人生阶段出问题：
  - 财被劫断 → 破财
  - 官被伤断 → 失职、官非
  - 印被财断 → 失学、失母
  - 食被枭断 → 失子、断财源
  - 身被官杀断 → 灾病、压力

## 四、男女命十神六亲细分

六亲取用以十神为本，男女有别。

### 1. 男命六亲取用表

| 六亲 | 十神 | 说明 |
|------|------|------|
| 父亲 | 偏财 | 偏财为父（克印者为父） |
| 母亲 | 正印 | 正印为母（生身者为母） |
| 兄弟 | 比肩 | 同我者为兄弟 |
| 姐妹 | 劫财 | 同我异阴者为姐妹 |
| 妻子 | 正财 | 我克者为妻，正财为正妻 |
| 情人/继室 | 偏财 | 偏财为偏房、情人 |
| 儿子 | 七杀 | （克日干者，男命以七杀为子，亦有以食伤为子者，流派不同） |
| 女儿 | 正官 | （亦有以食伤为女者） |
| 祖上 | 年柱财官印 | 看年柱十神 |
| 子女 | 时柱食官杀 | 看时柱十神 |

> 注：男命子女取用，书房派多取食伤（我生者为子女），盲派有取官杀者。实战以食伤为主、官杀为辅参考。

### 2. 女命六亲取用表

| 六亲 | 十神 | 说明 |
|------|------|------|
| 父亲 | 正财 | （女命以财为父，正财为父） |
| 母亲 | 偏印 | （女命以偏印为母，亦有取正印者） |
| 兄弟 | 比肩 | 同 |
| 姐妹 | 劫财 | 同 |
| 丈夫 | 正官 | 克我者为夫，正官为正夫 |
| 情人/继夫 | 七杀 | 七杀为偏夫、情人 |
| 儿子 | 伤官 | （我生者为子，女命以伤官为子） |
| 女儿 | 食神 | （食神为女） |
| 祖上 | 年柱 | 看年柱十神 |
| 子女 | 时柱食伤 | 看时柱十神 |

> 注：女命六亲取用流派分歧较大。主流以正官为夫、七杀为偏夫、食伤为子女、偏印为母、正财为父。

### 3. 六亲吉凶判断要点
- **六亲星为用神且无伤**：该六亲得力、有助、缘深。
- **六亲星为忌神且旺**：该六亲拖累、缘薄、多灾。
- **六亲星被冲克合化**：该六亲缘薄、分离、灾病。
- **六亲宫被冲克**：该宫位所主六亲不安（如日支被冲，配偶不稳）。
- **六亲星入墓逢空**：该六亲缘薄、早逝、分离。

## 五、十神真假、清浊、混杂

### 1. 真假
- **真神**：月令司令之神透干有力，为真用神。真神有力，格局高。
- **假神**：非月令司令之神，或虚浮无根，为假。假神无力，格局低。
- 真假之分决定格局层次。《子平真诠》重月令透干取格，即求真神。

### 2. 清浊
- **清**：用神单一不杂，无忌神混杂，或忌神有制。主一生纯粹、事业专注、富贵显。
- **浊**：用神被忌神混杂，或财官印多而杂。主一生反复、事业多变、富贵减。
- 如正官格见一官无杂为清，见官杀混杂为浊。

### 3. 混杂
- **官杀混杂**：正官七杀同现，且均有力无制。主事多反复、女命婚姻不稳、是非多。宜去杀留官或去官留杀，方清。
- **财星混杂**：正财偏财同现且旺。男命多情、婚姻不稳、财来财去。
- **印星混杂**：正印偏印同现。主学问杂而不精、母多（继母）、性格矛盾。
- **食伤混杂**：食神伤官同现。主才华多而不专、口舌多。

**去杂留清法**：
- 合去：如官杀混杂，以伤官合杀（丙辛合类）去杀留官。
- 制去：以食制杀、以印制伤。
- 冲去：以冲去其混杂者。
- 化去：以合化去其杂。

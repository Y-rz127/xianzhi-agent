# 社交人际断法

> 八字看社交人际、朋友同辈、贵人小人、合伙关系的专项断法规则。供社交人际 Worker 检索引用。

## 一、总则

1. **比劫看同辈朋友**：比肩劫财代表与日主同辈之人，为用则朋友得力，为忌则受朋友连累。
2. **贵人星看提携**：天乙、天德、月德等贵人入命，主社交中有贵人相助。
3. **官杀看小人压制**：七杀无制主小人，攻身则招嫉妒使坏；正官有制主贵人长辈。
4. **格局清浊看层次**：清格主贵人层次高、圈子优质；浊格主交际复杂、是非多。

## 二、比肩劫财与社交

### 1. 比肩
- **含义**：与日主同阴阳同五行，主同性朋友、同辈、同业。
- **为用**：朋友得力、重义气、合伙成事、人脉广。
- **为忌**：受朋友拖累、争利争名、合伙生隙、易被同辈压制。

### 2. 劫财
- **含义**：与日主异阴阳同五行，主异性朋友、竞争对手。
- **为用**：异性贵人得力、善交际、能争能抢。
- **为忌**：朋友夺财、合伙破财、口舌是非、易被异性连累。

### 3. 比劫夺财
- 比劫旺而夺财：不仅破财，也主朋友争利、合伙生隙。
- 比劫逢官杀制：朋友虽多但有约束，不致过耗。

## 三、贵人星与社交

### 1. 天乙贵人
- **查法**：日干查（甲戊庚丑未 / 乙己子申 / 丙丁亥酉 / 壬癸卯巳 / 辛午寅）。
- **作用**：主最高层次的贵人，遇难呈祥、逢凶化吉，社交中得大人物提携。

### 2. 天德贵人 / 月德贵人
- **作用**：主道德贵人，心地善良者易得长辈、上司赏识，社交中得正派之人助。

### 3. 三奇贵人
- **作用**：主奇才异能，社交中易遇非凡之人，自身也多才多艺。

## 四、七杀与小人

### 1. 七杀无制
- 七杀攻身无食伤制化：易招小人嫉妒、背后使坏、暗中算计。
- 七杀逢食神制：小人化为权柄，反主掌权驭人。

### 2. 伤官见官
- 伤官旺而见正官：易得罪上司、口舌招非、社交中树敌。
- 伤官配印：口才化为才华，反主以言得贵。

### 3. 桃花劫煞
- 桃花逢劫煞：社交中易因色招祸、朋友配偶生隙。

## 五、宫位与社交

### 1. 日支合他柱
- 日支被合走：身边人缘变化（合入为得助，合走为疏远）。
- 日支合财星/官星：社交中因配偶或财官得人脉。

### 2. 月柱看同辈
- 月柱比劫旺：兄弟姐妹、同辈朋友多。
- 月柱逢冲：与同辈关系动荡、早离家。

## 六、大运流年引动

- **走比劫旺地**：社交活跃、人脉变动、易合伙或争利。
- **走官杀旺地**：遇贵人或受压制，看官杀喜忌。
- **走食伤旺地**：以才情交友、口才得人，但也易口舌。
- **走印星旺地**：长辈贵人提携、学业圈人脉。
- **走财星旺地**：因财结缘、商务社交，但比劫也易夺财。

## 七、格局清浊与社交层次

- **清格**（用神有力、格局纯正）：贵人层次高、交往圈子优质。
- **浊格**（用神受伤、格局混杂）：交际复杂、是非多、层次参差。
- **从格**：交际圈子单一但极致（从财主财圈、从官主官圈）。

## 八、合伙择友参考

1. 比劫为用、格局清者，合伙可成。
2. 比劫为忌、劫财夺财者，忌合伙，独立经营为佳。
3. 贵人星入命、用神有力者，合伙易遇贵人。
4. 七杀无制、伤官见官者，合伙易生是非、忌长期绑定。

## 九、合规口径

1. 社交人际分析仅为命理趋势参考，不替代现实人际判断与心理辅导。
2. 不绝对断"此人必为小人""此人必为贵人"，应说明判定依据与变数。
3. 涉及人际冲突，劝导以沟通与现实手段化解，不宣扬符咒化解等。

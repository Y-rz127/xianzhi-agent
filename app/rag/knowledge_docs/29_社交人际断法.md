# 社交人际断法

> 八字看社交人际、朋友同辈、贵人小人、合伙关系、职场社交的专项断法规则。供社交人际 Worker 检索引用。

## 一、总则

1. **比劫看同辈朋友**：比肩劫财代表与日主同辈之人，为用则朋友得力，为忌则受朋友连累。
2. **贵人星看提携**：天乙、天德、月德等贵人入命，主社交中有贵人相助。
3. **官杀看小人压制**：七杀无制主小人，攻身则招嫉妒使坏；正官有制主贵人长辈。
4. **格局清浊看层次**：清格主贵人层次高、圈子优质；浊格主交际复杂、是非多。

## 二、比肩劫财与社交

### 1. 比肩
- **含义**：与日主同阴阳同五行，主同性朋友、同辈、同业。
- **为用**：朋友得力、重义气、合伙成事、人脉广。
- **为忌**：受朋友拖累、争利争名、合伙生隙、易被同辈压制。

### 2. 劫财
- **含义**：与日主异阴阳同五行，主异性朋友、竞争对手。
- **为用**：异性贵人得力、善交际、能争能抢。
- **为忌**：朋友夺财、合伙破财、口舌是非、易被异性连累。

### 3. 比劫夺财
- 比劫旺而夺财：不仅破财，也主朋友争利、合伙生隙。
- 比劫逢官杀制：朋友虽多但有约束，不致过耗。

### 4. 比劫与社交风格
- **比肩旺**：社交中重义气、讲兄弟情、圈子以同性好友为主。
- **劫财旺**：社交中善交际、异性缘好、但易争风吃醋。
- **比劫均旺**：人脉广但杂、朋友多但真心少、社交消耗大。
- **比劫弱**：朋友少、不善社交、独来独往、但朋友质量高。

## 三、十神组合与社交风格

### 1. 食伤旺者
- **社交特点**：口才好、善表达、以才情交友、社交中是焦点。
- **优势**：人缘好、容易成为社交中心、擅长演讲和说服。
- **风险**：言多必失、口舌招非、容易得罪人。
- **建议**：学会倾听、控制表达欲、三思而后言。

### 2. 印星旺者
- **社交特点**：内敛含蓄、重精神交流、圈子小而精。
- **优势**：得长辈贵人提携、学术圈人脉、社交中给人信赖感。
- **风险**：社交被动、不善主动拓展、容易孤立。
- **建议**：适当参加社交活动、主动表达善意。

### 3. 官杀旺者
- **社交特点**：有威严、重规矩、社交中给人距离感。
- **优势**：得上级赏识、职场人脉、社交中有话语权。
- **风险**：给人压迫感、不易亲近、社交面窄。
- **建议**：适当放下身段、培养亲和力。

### 4. 财星旺者
- **社交特点**：务实社交、因财结缘、商务人脉广。
- **优势**：社交中有资源、商务合作机会多、人脉变现能力强。
- **风险**：社交功利化、酒肉朋友多、真情少。
- **建议**：区分商务社交与私人友谊、培养非功利性友谊。

## 四、贵人星与社交

### 1. 天乙贵人
- **查法**：日干查（甲戊庚丑未 / 乙己子申 / 丙丁亥酉 / 壬癸卯巳 / 辛午寅）。
- **作用**：主最高层次的贵人，遇难呈祥、逢凶化吉，社交中得大人物提携。
- **入何宫**：年柱主祖上贵人、月柱主事业贵人、日支主配偶贵人、时柱主晚年贵人。

### 2. 天德贵人 / 月德贵人
- **作用**：主道德贵人，心地善良者易得长辈、上司赏识，社交中得正派之人助。

### 3. 三奇贵人
- **作用**：主奇才异能，社交中易遇非凡之人，自身也多才多艺。

### 4. 文昌贵人
- **作用**：主聪明好学，社交中易遇文化圈、学术圈贵人。

### 5. 贵人入命与社交层次
- 天乙+天德同入：贵人层次极高，社交中得大人物双重提携。
- 贵人逢冲：贵人力量减弱，提携不稳。
- 贵人逢合：贵人力量增强，提携稳固。
- 贵人坐空亡：贵人虚浮、有贵人之名无其实。

## 五、七杀与小人

### 1. 七杀无制
- 七杀攻身无食伤制化：易招小人嫉妒、背后使坏、暗中算计。
- 七杀逢食神制：小人化为权柄，反主掌权驭人。

### 2. 伤官见官
- 伤官旺而见正官：易得罪上司、口舌招非、社交中树敌。
- 伤官配印：口才化为才华，反主以言得贵。

### 3. 桃花劫煞
- 桃花逢劫煞：社交中易因色招祸、朋友配偶生隙。

### 4. 识别小人方法
- 七杀攻身：明面上的对手、公开竞争。
- 伤官见官：暗中的口舌、背后议论。
- 劫财夺财：以朋友面目出现、实则争利。
- 枭神夺食：表面关心、实则断你财路。

### 5. 化解小人
- 食神制杀：以智取胜、以才干压住小人。
- 印星化杀：以德服人、以贵人制小人。
- 比劫抗杀：以团队力量对抗。
- 财星生官：以资源化解矛盾。

## 六、宫位与社交

### 1. 日支合他柱
- 日支被合走：身边人缘变化（合入为得助，合走为疏远）。
- 日支合财星/官星：社交中因配偶或财官得人脉。

### 2. 月柱看同辈
- 月柱比劫旺：兄弟姐妹、同辈朋友多。
- 月柱逢冲：与同辈关系动荡、早离家。

### 3. 年柱看长辈
- 年柱印星为用：得祖上/长辈荫蔽、社交中有长辈贵人。
- 年柱逢冲：与长辈关系疏远、早年社交受挫。

### 4. 时柱看晚辈
- 时柱食伤为用：晚辈得力、学生/下属有出息。
- 时柱逢空亡：与晚辈缘薄、晚年社交圈缩小。

## 七、大运流年引动

- **走比劫旺地**：社交活跃、人脉变动、易合伙或争利。
- **走官杀旺地**：遇贵人或受压制，看官杀喜忌。
- **走食伤旺地**：以才情交友、口才得人，但也易口舌。
- **走印星旺地**：长辈贵人提携、学业圈人脉。
- **走财星旺地**：因财结缘、商务社交，但比劫也易夺财。
- **走桃花旺地**：异性缘增强、社交中桃花事件多。
- **走驿马旺地**：社交圈变动、异地人脉、出差社交。

## 八、格局清浊与社交层次

- **清格**（用神有力、格局纯正）：贵人层次高、交往圈子优质。
- **浊格**（用神受伤、格局混杂）：交际复杂、是非多、层次参差。
- **从格**：交际圈子单一但极致（从财主财圈、从官主官圈）。

## 九、合伙择友参考

### 1. 适合合伙
- 比劫为用、格局清者：合伙可成。
- 贵人星入命、用神有力者：合伙易遇贵人。
- 食伤生财为用：以技能合伙、各展所长。
- 财官双美：合伙名利双收。

### 2. 不宜合伙
- 比劫为忌、劫财夺财者：忌合伙，独立经营为佳。
- 七杀无制、伤官见官者：合伙易生是非、忌长期绑定。
- 财星被冲、比劫争财者：合伙必争利、散伙收场。
- 正偏财混杂者：合伙方向不专、易分心。

### 3. 合伙人选择
- 用神互补：双方用神五行互补，合伙最吉。
- 忌神相同：双方忌神相同，低谷时互相拖累。
- 比劫为用者：宜找比劫型合伙人（执行力强）。
- 食伤为用者：宜找食伤型合伙人（创意策划）。

## 十、职场社交 vs 朋友社交

### 1. 职场社交
- 看官杀与印星：官印相生者职场人脉好、得上级提携。
- 伤官见官者：职场社交易冲突、与上级不合。
- 食伤生财者：以业绩和才干获得职场认可。
- 比劫夺财者：职场竞争激烈、同事争利。

### 2. 朋友社交
- 看比劫与食伤：比劫为用者朋友多且得力。
- 食伤旺者：朋友圈广、善交际、人缘好。
- 印星旺者：朋友少但深交、重精神交流。
- 财星旺者：朋友多因利益结缘、酒肉朋友多。

## 十一、社交应期参考

1. 大运走比劫旺地 + 流年引动比劫 → 社交活跃期、人脉拓展期。
2. 大运走官杀旺地 + 流年透官印 → 贵人提携期、职场升迁期。
3. 大运走食伤旺地 + 流年引动食伤 → 才华展示期、以言得贵期。
4. 大运走七杀攻身 + 流年引动伤官 → 小人作祟期、口舌是非期。
5. 大运走驿马旺地 + 流年冲动日支 → 社交圈变动期、异地人脉期。

## 十二、合规口径

1. 社交人际分析仅为命理趋势参考，不替代现实人际判断与心理辅导。
2. 不绝对断"此人必为小人""此人必为贵人"，应说明判定依据与变数。
3. 涉及人际冲突，劝导以沟通与现实手段化解，不宣扬符咒化解等。
4. 涉及职场霸凌、人际伤害，劝导寻求法律和心理专业帮助。

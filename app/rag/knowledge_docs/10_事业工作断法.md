# 事业工作断法

> 八字看事业工作、职业方向、升职降职、跳槽创业、职场人际的专项断法规则。供事业工作 Worker 检索引用。
> 引用典籍：《渊海子平·论财官》《子平真诠·论格局》《滴天髓·论官杀》。

## 一、事业判断核心

事业工作优先看官杀、印星、食伤与日主强弱的配合。

- **官杀**：规则、职位、压力、上级、平台约束。官杀清透且有印化，主稳定平台、职级晋升、专业资质路线。
- **食伤**：表达、技术输出、产品能力、自由度。食伤生财时，适合用技能、内容、产品、销售转化收入。
- **印星**：学习、证书、贵人、制度资源。日主偏弱时，印星能承压，适合先借平台和资质发展。
- **比劫**：同辈、竞争、合伙、团队。比劫旺而财弱时，合伙与分钱问题要谨慎。
- **财星**：收入、资源、目标。财为养命之源，官为荣身之本。

## 二、八格事业适配

不同格局，事业方向适配不同：

| 格局 | 事业适配 | 不宜 |
|------|----------|------|
| 正官格 | 公职、行政、管理、体制内、大型机构稳定岗 | 自由散漫、抗上越权 |
| 七杀格 | 武职、军警、执法、开拓型岗位、危机管理 | 文牍清闲、被动执行 |
| 正财格 | 稳定薪资、财务、银行、实业经营、守成 | 投机冒险、横财 |
| 偏财格 | 经商、贸易、投资、销售、外交、流通业 | 死工资、坐班固定 |
| 正印格 | 文职、教育、学术、文化、宗教、研究、文秘 | 商业争利、投机 |
| 偏印格 | 偏门、玄学、技术专才、设计、五术、科研 | 大众管理、销售 |
| 食神格 | 文艺、餐饮、教育、服务、医护、福利、子女相关 | 严肃执法、争权 |
| 伤官格 | 技艺、演艺、销售、写作、发明、自由职业、营销 | 体制内守规矩、服从 |

## 三、十神组合与职业细分

### 1. 官印相生
- **特征**：正官配正印，官星有印化，职位稳固、有靠山。
- **适配**：公务员、事业单位、国企管理、教育行政、医院管理。
- **升职路径**：稳扎稳打、按资排辈、逢印比大运升迁。

### 2. 杀印相生
- **特征**：七杀配印星，杀有制化，谋略深远、掌权驭人。
- **适配**：军警、企业高管、创业者、危机管理、战略规划。
- **升职路径**：破格提拔、临危受命、逢杀印大运掌权。

### 3. 食神制杀
- **特征**：食神制七杀，以智勇压住压力，魄力与才干兼备。
- **适配**：技术管理、项目经理、创业实干、执法一线。
- **升职路径**：凭能力上位、逢食伤制杀运得权。

### 4. 伤官配印
- **特征**：伤官配印，才华内敛、学术与创意兼备。
- **适配**：科研、设计、策划、咨询、专业技术、教育培训。
- **升职路径**：凭专业实力晋升、逢印星大运得认可。

### 5. 食伤生财
- **特征**：食伤生财，以技艺、内容、产品变现。
- **适配**：自媒体、内容创作、技术外包、销售、餐饮、服务业。
- **升职路径**：不靠职级靠收入，逢食伤生财运进财。

### 6. 财官双美
- **特征**：财星生官、官星护财，名利双收。
- **适配**：金融、投资、企业主、高管、商务。
- **升职路径**：逢财官大运名利双收。

### 7. 比劫夺财
- **特征**：比劫旺而夺财，竞争激烈、分财分利。
- **适配**：竞争性行业、销售、体育、独立经营。
- **注意**：不宜合伙、不宜固定薪资，宜凭个人能力单打独斗。

## 四、行业五行对应

| 五行 | 行业方向 |
|------|----------|
| 木 | 教育、文化、出版、医药、服装、农林、环保、设计 |
| 火 | 电子、IT、传媒、能源、餐饮、美容、照明、军工 |
| 土 | 房地产、建筑、农业、矿业、仓储、殡葬、中介 |
| 金 | 金融、机械、汽车、珠宝、法律、军警、外科 |
| 水 | 航运、物流、旅游、通讯、饮料、渔业、咨询、贸易 |

### 用神五行选行业原则

- 用神为木：宜从事木属性行业，或东方城市发展。
- 用神为火：宜从事火属性行业，或南方城市发展。
- 用神为土：宜从事土属性行业，或本地/中原发展。
- 用神为金：宜从事金属性行业，或西方城市发展。
- 用神为水：宜从事水属性行业，或北方城市发展。
- 忌神行业宜避，但忌神被制化时可酌情。

## 五、跳槽、裸辞、创业细分断法

### 1. 跳槽
- 问跳槽不只看流年，还要看当前大运是否支持职业变化。
- 流年冲动月柱、官杀、事业相关十神时，易出现岗位变化、领导调整、组织重组。
- 食伤旺而财星被引动，适合主动争取机会、谈资源、做项目转化。
- 官杀压力大而印星不足时，不建议裸辞，宜先确定岗位、直属领导、合同和现金流。
- 大运行喜用神运：跳槽得利、升迁。
- 大运行忌神运：跳槽多败，宜稳守。

### 2. 裸辞
- 财星有根、食伤生财、印比护身：可裸辞过渡，再寻良机。
- 财星虚浮、官杀克身、比劫夺财：不宜裸辞，断流则困。
- 七杀旺无制、羊刃冲克：裸辞多灾，宜骑驴找马。

### 3. 合伙创业
- 看财星、食伤、比劫和大运承载力。
- 财星被冲或比劫夺财明显时，投资和合伙要保守。
- 比劫旺为用：合伙得力，团队互助。
- 比劫旺为忌：合伙争财、内讧分赃，不宜合伙。
- 食伤生财为用：创业有成，以技艺致富。
- 官杀混为忌：创业多官非，宜守不宜创。

### 4. 副业
- 食伤旺为用：副业有成，多源收入。
- 印旺克食伤：副业转化不足，宜先提升技能。
- 比劫旺：副业宜合伙分摊。

## 六、升职降职断法

### 1. 升职信号
- 大运走官杀旺地、印星生扶：升职良机。
- 流年透官星、印星：当年有升职机会。
- 食伤制杀得力：凭能力上位。
- 财官相生：因业绩或资源得提拔。
- 官印相生逢生扶：稳升、有靠山。

### 2. 降职信号
- 大运走伤官见官：降职、与上级冲突。
- 流年官杀被克：丢权、降职。
- 比劫夺官：被同辈挤掉。
- 印星受克：失靠山、降职。
- 七杀攻身无制：被压制、降职。

### 3. 失业信号
- 官杀被克尽：失业。
- 财星被劫尽：收入断流。
- 伤官见官+比劫夺财：失业+破财。
- 大运走忌神、流年又冲克用神：失业风险最高。

## 七、职场人际断法

### 1. 与上级关系
- 官杀为用且有印化：上级赏识、关系融洽。
- 伤官见官：与上级冲突、不服管。
- 七杀攻身无制：上级压制、职场霸凌。
- 印星生扶：得上级提携、有靠山。

### 2. 与同事关系
- 比劫为用：同事互助、团队和谐。
- 比劫为忌：同事竞争、争利争名。
- 食伤泄秀：以才干服人、同事认可。
- 官杀制比劫：有规矩约束、同事不致过分。

### 3. 与下属关系
- 食伤为用：善于培养下属、团队出成绩。
- 七杀制比劫：管得住人、有威严。
- 比劫旺无制：下属不服、团队散漫。
- 印星太旺：过度包办、下属依赖。

## 八、事业应期参考

1. 大运走官杀旺地 + 流年透官印 → 升职应期。
2. 大运走食伤生财 + 流年引动财星 → 创业进财应期。
3. 大运走伤官见官 + 流年冲克官星 → 降职/跳槽应期。
4. 大运走比劫夺财 + 流年劫财 → 破财/失业应期。
5. 大运走印星生扶 + 流年官印相生 → 升职/考证应期。

## 九、合规口径

1. 事业分析仅为命理趋势参考，不替代职业规划与现实判断。
2. 不绝对断"必定升职""必定失业"，应说明判定依据与变数。
3. 涉及辞职、创业等重大决策，劝导综合现实因素（现金流、家庭、市场）。
4. 不宣扬符咒改运、风水改运等非理性手段。

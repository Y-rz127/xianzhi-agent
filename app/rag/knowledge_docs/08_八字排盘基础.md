# 八字排盘基础

## 四柱
八字由年柱、月柱、日柱、时柱组成，每柱一天干一地支，共八字。

## 年柱
以立春为年界（非农历正月初一，更非公历元旦）。
年柱 = 当年干支。

## 月柱
月柱地支固定：寅为正月、卯二月、辰三月、巳四月、午五月、未六月、申七月、酉八月、戌九月、亥十月、子十一月、丑十二月。
月柱天干由年干推算（五虎遁）：
- 甲己之年丙作首（甲己年正月起丙寅）
- 乙庚之岁戊为头
- 丙辛之岁寻庚上
- 丁壬壬寅顺水流
- 戊癸何方发，甲寅之上好追求

## 日柱
日柱干支由万年历查询，无简单推算公式。日柱为命主，最重要。

## 时柱
时柱地支：23-1 子时，1-3 丑时，3-5 寅时，5-7 卯时，7-9 辰时，9-11 巳时，11-13 午时，13-15 未时，15-17 申时，17-19 酉时，19-21 戌时，21-23 亥时。
时柱天干由日干推算（五鼠遁）：
- 甲己还加甲（甲己日之子时起甲子）
- 乙庚丙作初
- 丙辛从戊起
- 丁壬庚子居
- 戊癸何方发，壬子是真途

## 注意
- 子时（23-1）分早子时（0-1）和夜子时（23-0），有争议
- 节气为月界，立春为年界
- 排盘必须精确到时辰，差一时辰则时柱不同
